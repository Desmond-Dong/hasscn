# Home Assistant 前端

Home Assistant 前端让用户能够浏览并控制家中状态、管理自动化，并配置集成。

前端以移动优先的体验为设计目标。它是一个渐进式 Web 应用，为用户提供接近原生应用的使用体验。

Home Assistant 前端需要足够快，但也必须能在大量较旧的设备上运行。为此，我们提供两个版本的前端：

* **Latest：** 这个构建兼容主流浏览器最近两个版本，并针对性能进行了优化。
* **ES5：** 这个构建兼容过去 5 年以上发布的浏览器，并针对兼容性进行了优化。

能够运行最新技术的设备并不一定足够快。你可以买到运行最新版 Android、可使用最新版 Firefox 和 Chrome 浏览器的入门级 Android 手机，但它们可能配备低性能芯片组且内存有限。我们的 Latest 构建也需要在这些设备上流畅运行。

如果想深入了解前端及其设计选择，请参阅[这篇博客文章](https://www.home-assistant.io/blog/2019/05/22/internet-of-things-and-the-modern-web)。
