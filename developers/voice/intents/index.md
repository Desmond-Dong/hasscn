# 意图

有关意图，请参阅 Home Assistant Core 文档中的以下内容：

* Home Assistant 支持的[内置意图](/developers/intent_builtin.md)。
* 使用[对话 API](/developers/intent_conversation_api.md) 向 Home Assistant 发送文本命令并接收响应。
* [注册意图处理器](/developers/intent_handling.md)，为 Home Assistant 添加新的意图。
* 如果你有自己的句子匹配逻辑并希望由 Home Assistant 处理意图，可使用[触发意图](/developers/intent_firing.md)。
