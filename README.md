# Home Assistant 中文网

## 🚀 在全新环境下运行此网站所需的命令

### 1. 克隆项目（如果还没有）
```bash
git clone https://github.com/ha-china/hasscn.top.git
cd hasscn.top
```

### 2. 安装依赖
```bash
npm install
```

### 3. 启动开发服务器
```bash
npm run dev
```
网站将在 http://localhost:3000 启动

### 4. 构建生产版本（可选）
```bash
npm run build
```

### 5. 预览生产构建（可选）
```bash
npm run preview
```

## 📋 环境要求

- Node.js 16+
- npm 或 yarn

## 🌟 项目特性

- ✅ 完整的双语支持（中文/英文）
- ✅ 响应式设计，支持各种屏幕尺寸
- ✅ 优化的阅读体验，宽屏内容区域达到1400px
- ✅ 首页图标精美设计，带悬停动画效果
- ✅ Home Assistant 官方主题色彩方案
- ✅ 支持深色/浅色模式切换

## 📝 项目说明

这是 Home Assistant 的中文网站，为中文用户提供：
- 🏠 Home Assistant OS 极速版介绍和下载
- 📚 官方文档中文本地化
- 🛠️ 加载项和集成推荐
- 💡 技术支持和教程

开发服务器启动后，您可以在浏览器中访问 http://localhost:3000 查看网站。
