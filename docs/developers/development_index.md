# Home Assistant 核心

Home Assistant Core 是从零构建的，并且可以通过集成轻松扩展。本节重点介绍如何开发集成。

在开始之前，请先阅读 [Home Assistant 架构](/developers/architecture_index) 总览页面，以便熟悉构成 Home Assistant 的核心概念。

如果你在开发过程中需要帮助，或对如何参与贡献有疑问，请加入 [Discord 社区](https://www.home-assistant.io/join-chat/)，进入 **#developers** 频道，或在 **#support** 中创建主题。

若要获取开发相关频道的访问权限，请在侧边栏中选择“**频道和身份组**”，然后启用“**我想为 Home Assistant 贡献开发技能**”这一身份组。
