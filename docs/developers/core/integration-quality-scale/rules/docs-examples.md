## 推理

为了展示如何使用集成，请提供一组有限的蓝图，其中包含常见或有用的蓝图。
这将帮助用户更快、更轻松地开始集成。

将蓝图上传到 [`source/blueprints/integrations`](https://github.com/home-assistant/home-assistant.io/tree/current/source/blueprints/integrations) 下的蓝图文件夹，或上传到论坛上的 [blueprint exchange](https://community.home-assistant.io/c/blueprints-exchange)。在集成页面上，添加指向蓝图的链接。

请勿将文档页面用作集合或蓝图文件夹或蓝图交换的替代品。
