我们维护了一个设计门户 [https://design.home-assistant.io](https://design.home-assistant.io)，其中提供了前端多个方面的信息，例如：

* 可复用的 UI 组件。
* 各种配置和状态下的 Dashboard cards。
* 轻松比较浅色与深色主题下的组件和 cards。
* 关于 Home Assistant 中特定措辞的说明。

当前端中新增组件或功能时，也需要将它们加入设计门户。关于具体做法，请参见这个门户页面：[https://design.home-assistant.io/#design.home-assistant.io/editing](https://design.home-assistant.io/#design.home-assistant.io/editing)

:::note
虽然这个门户对外公开名称是 "design"，但在前端仓库中它被称为 "gallery"。因此，在本地开发环境中运行 gallery 的脚本位于 `gallery/script/develop_gallery`，源代码位于 `gallery/src`。
:::
