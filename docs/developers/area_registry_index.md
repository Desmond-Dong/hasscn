# 区域注册表

区域注册表是 Home Assistant 用于跟踪区域的注册表。区域表示 Home Assistant 中的一个物理位置。它可用于将设备放置到不同区域中。

| Attribute | Description |
| --------- | ----------- |
| id | 区域的唯一 ID（由 Home Assistant 生成） |
| name | 此区域的名称 |
