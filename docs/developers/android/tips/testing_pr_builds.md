# 测试拉取请求构建

您可以通过下载 CI 生成的 APK 工件来测试拉取请求中的更改，而无需在本地进行构建。

1. 转到 GitHub 上的拉取请求。
2. 单击**支票** 选项卡。
3. 在左侧边栏中，单击**请求请求**。
4. 滚动到页面底部以找到**文物** 部分。
5. 下载 **APK** 工件（包含所有 APK 变体的 ZIP 文件）。
6. 解压 ZIP 并找到 `full` 或 `minimal` APK。
7. 在您的设备上安装 APK：
   - **从电话里**：直接在手机上下载工件并使用文件管理器应用程序进行安装。
   - **使用亚行**：从您的计算机运行`adb install path/to/app.apk`。

:::note
您必须登录 GitHub 帐户才能下载工件。 CI 工作流程还必须成功完成，APK 可供下载。
:::

:::note
调试 APK 与应用程序的生产版本一起安装，允许您同时安装两者。调试版本有一个红色图标，以将其与生产应用程序区分开来。
:::
