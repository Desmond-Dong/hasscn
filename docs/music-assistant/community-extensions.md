# 概述 <img src="/assets/icons/community-extensions-icon.png" alt="预览图片" style="width: 70px; float: right;"  loading="lazy" />
这是社区开发的 Music Assistant HA 卡片或仪表板集合。请查看链接的仓库（仓库标题已链接）以获取这些项目的说明和状态。这些项目的复杂程度各不相同。
***************************************************************
## <a href="https://github.com/DanStennett/HAMusicAssistantJukebox" target="_blank" rel="noopener noreferrer">Music Assistant 点唱机</a>

一个基于 Web 的歌曲请求系统，与 Home Assistant 和 Music Assistant 集成，为客人创造互动点唱机体验！

<img src="https://github.com/DanStennett/HAMusicAssistantJukebox/blob/main/readme_image.jpg?raw=true" alt="图片" style="width: 420px;"  loading="lazy" />

***************************************************************
## <a href="https://github.com/fastxl2024/mass-search-card" target="_blank" rel="noopener noreferrer">Music Assistant 搜索卡片</a>

Mass Search Card 是 Home Assistant 的高级搜索卡片，旨在简化与 Music Assistant 的交互。此卡片允许您轻松搜索艺术家、曲目、专辑、播放列表和电台，并在选定的媒体播放器上播放。

## <a href="https://github.com/droans/mass-player-card" target="_blank" rel="noopener noreferrer">Music Assistant 播放器卡片</a>
