# 获取帮助 <img src="/assets/icons/support-icon.png" alt="预览图片" style="width: 70px; float: right;"  loading="lazy" />

由于本项目源于 Home Assistant 集成，所有最终用户支持都在单独的仓库中维护。请<a href="https://github.com/music-assistant/support" target="_blank" rel="noopener noreferrer">在此</a>提交问题并讨论想法

直接链接：

- <a href="https://github.com/music-assistant/support/issues" target="_blank" rel="noopener noreferrer">问题跟踪器</a>用于创建错误报告。请先查看[故障排除](/music-assistant/faq/troubleshooting/)页面，并检查您的问题是否已被报告。提交报告时请包含详细信息和日志文件。
- <a href="https://github.com/orgs/music-assistant/discussions/categories/feature-requests-and-ideas" target="_blank" rel="noopener noreferrer">功能请求</a>：为现有请求投票、参与讨论或添加新请求。
- <a href="https://github.com/orgs/music-assistant/discussions/categories/q-a" target="_blank" rel="noopener noreferrer">问答区</a>常见问题解答和教程

此外，您还可以在<a href="https://discord.gg/kaVm8hGpne" target="_blank" rel="noopener noreferrer">Discord 社区</a>寻求帮助。加入社区获取支持！

**提问之前：**

查看如何<a href="http://www.catb.org/esr/faqs/smart-questions.html" target="_blank" rel="noopener noreferrer">聪明地提问</a>

在我们的任何支持渠道提出技术问题之前，请执行以下操作：

1. 尝试通过搜索本文档和<a href="https://github.com/orgs/music-assistant/discussions" target="_blank" rel="noopener noreferrer">讨论区</a>找到答案
2. 尝试通过搜索<a href="https://github.com/music-assistant/support/issues?q=is%3Aissue+is%3Aclosed" target="_blank" rel="noopener noreferrer">已关闭的问题</a>找到答案
3. 尝试通过网络搜索找到答案：https://www.google.com/
4. 尝试通过检查或实验找到答案
5. 尝试向有经验的朋友请教

当您提问时，请表明您已经完成了以上工作；这将有助于证明您不是懒惰的人，没有浪费他人的时间。更好的是，展示您从这些工作中学到了什么。我们喜欢为那些证明自己能够从答案中学习的人解答问题。
