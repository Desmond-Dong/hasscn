# Home Assistant 集成 <img src="/assets/icons/ha-logo.png" alt="预览图片" style="width: 70px; float: right;"  loading="lazy" />

Music Assistant 的 Home Assistant 集成提供从 MA 到 HA 的连接。这意味着 MA 播放器在 HA 中可见，可以通过 HA UI 或通过自动化或脚本进行控制。

- 完全支持来自 HA 的[公告](/music-assistant/integration/announcements/)，形式为文本转语音或音频文件。

- 通过一些额外设置，通过 HA 的[MA 语音控制](/music-assistant/integration/voice/)也是可能的

- 集成可以连接到作为 HA 插件或作为同一主机或另一主机系统上的 docker 容器运行的 MA 服务器

- 该集成是 Home Assistant 的官方部分，无需安装自定义组件

另请参阅 <a href="https://www.home-assistant.io/integrations/music_assistant/" target="_blank" rel="noopener noreferrer">Home Assistant 关于 Music Assistant 集成的文档</a>

请注意，MA 内还有一个 [Home Assistant 提供者](/music-assistant/player-support/ha/)，其功能正好相反。它在 Music Assistant 中启用 Home Assistant 功能（和播放器）！
