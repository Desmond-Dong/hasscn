# 翻译

如果您关注前端仓库，您可能已经看到我们现在使用 Lokalise 作为中央翻译管理门户。

<a href="https://lokalise.com" target="_blank" rel="noopener noreferrer"><img src="https://github.com/lokalise/i18n-ally/raw/screenshots/lokalise-logo.png?raw=true" alt="Lokalise logo" width="275px" loading="lazy"></a>

在撰写本文时，MA 中各种语言的翻译状态如下

!<a href="/assets/screenshots/lokalise-stats.png" target="_blank"><img src="/assets/screenshots/lokalise-stats.png" alt="image" loading="lazy" style="max-width: 100%;" /></a>

当翻译百分比超过 75% 时，语言将被添加到前端。

我们很想支持更多语言，但我们需要您的帮助。

## 我该如何贡献？

- 在<a href="https://github.com/music-assistant/support/issues" target="_blank" rel="noopener noreferrer">问题跟踪器</a>中创建一个问题，说明您想贡献的语言。

- 然后我们将此语言添加到 Lokalise 的项目中，并在 GitHub 中解决该问题

- 问题解决后，您将能够使用此<a href="https://app.lokalise.com/public/321419046440c0f2bba414.05634832/" target="_blank" rel="noopener noreferrer">注册链接</a>作为翻译人员加入项目。

在 Lokalise 上请求加入团队时，请仅选择您打算贡献的语言。

!<a href="/assets/screenshots/lokalise-select-language.png" target="_blank"><img src="/assets/screenshots/lokalise-select-language.png" alt="image" loading="lazy" style="max-width: 100%;" /></a>

感谢您花时间阅读此文，我们期待您的贡献。
