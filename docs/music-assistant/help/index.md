# 我想帮忙 <img src="/assets/icons/help-icon.png" alt="Preview image" style="width: 70px; float: right;"  loading="lazy" />

对于这样一个大项目，总有很多事情要做。不仅是实际的代码编写，还有文档、提供支持、测试等。当然，您可以通过<a href="https://www.nabucasa.com" target="_blank" rel="noopener noreferrer">订阅 Nabu Casa</a>来大大帮助我们，但也可以通过其他方式帮助：

- 如果您想帮助开发，例如实现新的音乐提供商或修复特定播放器的怪癖，请在 Discord 上联系我们。还没有时间编写扩展的开发文档，但一旦您掌握了结构，它相对简单。当前文档可以在 <a href="https://developers.music-assistant.io/" target="_blank" rel="noopener noreferrer">Music Assistant Developer</a> 找到。还有<a href="https://github.com/music-assistant/server/tree/dev/music_assistant/providers" target="_blank" rel="noopener noreferrer">示例提供商</a>
- 提供或更新[翻译](/music-assistant/help/lokalise/)
- 在 Discord 或 GitHub 讨论部分帮助其他人
- 帮助编写文档和 HOWTO
- 帮助 <a href="https://github.com/OzGav" target="_blank" rel="noopener noreferrer">OzGav</a> 作为 Discord 和 GitHub 的版主，处理大量涌入的报告、请求和问题
- 确保通过点击 ⭐ 按钮<a href="https://github.com/music-assistant/server" target="_blank" rel="noopener noreferrer">喜欢这个项目</a>并与他人分享！

## 捐赠

如上所述，您可以以各种方式表达您的感谢。如果您想在财务上支持项目，您应该通过订阅 <a href="https://www.nabucasa.com/" target="_blank" rel="noopener noreferrer">Nabu Casa</a> 来支持 <a href="https://www.openhomefoundation.org/" target="_blank" rel="noopener noreferrer">Open Home Foundation (OHF)</a>。您也可以通过购买 <a href="https://www.home-assistant.io/" target="_blank" rel="noopener noreferrer">Home Assistant 硬件</a>或<a href="https://store.openhomefoundation.org/" target="_blank" rel="noopener noreferrer">周边产品</a>来支持 OHF！

如果您欣赏特定组件，请转到这些文档中的组件页面，找到负责的开发者，您可以看看他们是否有 Buy Me A Coffee 或类似的链接。主页上也有主要开发者 GitHub 页面的链接。[/music-assistant/#the-core-team]

整个团队提前表示非常感谢！
