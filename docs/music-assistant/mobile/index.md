# 移动设备

## 通知屏幕支持

当通过支持 <a href="https://caniuse.com/?search=mediasession" target="_blank" rel="noopener noreferrer">mediasession API</a> 的浏览器访问 Music Assistant UI 时，设备的通知或锁定屏幕上会显示媒体播放器小部件。支持暂停、播放、下一曲和上一曲功能。

虽然在使用 iOS HA 应用访问 Music Assistant UI 时已测试此功能可正常工作，但 Android HA 应用则不然。如果希望在 Android 设备上获得此功能，请使用支持的浏览器。

Android 用户还应注意，在远程设备上观察播放时，可能会看到"此手机"标签。这是 Android 操作系统的限制。

如果看不到媒体播放器小部件，则需要使用其他支持的浏览器。
