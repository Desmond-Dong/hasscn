## 功能

- 原生音频播放 - 通过 Sendspin 协议实现高质量音频输出，支持设备选择
- 系统托盘集成 - 从系统托盘控制播放并查看正在播放的内容
- 操作系统媒体控制 - 与 macOS 控制中心、Windows 媒体控制和 Linux MPRIS 集成
- Discord Rich Presence - 在 Discord 上显示您正在收听的内容
- 服务器发现 - 通过 mDNS 自动发现 Music Assistant 服务器

### <a href="https://discord.com/developers/docs/rich-presence/how-to#so-what-is-it" target="_blank" rel="noopener noreferrer">Discord Rich Presence</a>

与 Spotify 应用一样，Music Assistant 应用支持 Discord Rich Presence。
