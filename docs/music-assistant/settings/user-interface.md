# 用户界面设置 <img src="/assets/icons/UI-icon.png" alt="预览图片" style="width: 70px; float: right;"  loading="lazy" />

从此页面自定义用户界面

## 通用设置

- <b>主题。</b> 允许选择浅色或深色主题。选项有`自动（跟随系统配置）[默认]`、`浅色`和`深色`
- <b>语言。</b> 允许选择浅色或深色主题。选项有`自动（跟随系统配置）[默认]`或各种语言。翻译由[社区贡献](/music-assistant/help/lokalise/)。
- <b>启用的菜单项。</b> 可以显示的菜单项在此配置。选择项目的顺序就是它们显示的顺序。
- <b>强制移动布局。</b> 移动布局的菜单在底部，有 HOME 和 SEARCH 的图标，其余项目通过单个库图标访问
 
## Web 播放器

- <b>Web 播放器模式。</b> 选项有`Sendspin（实验性）[默认]`和`禁用（仅控制）`
- <b>Sendspin 同步延迟 (ms)。</b> 以毫秒为单位的静态延迟，用于调整音频同步。正值延迟播放，负值提前播放。使用此选项补偿设备特定的音频延迟。
- <b>使用浏览器延迟检测。</b> 使用浏览器的输出延迟 API 自动补偿硬件音频延迟（例如蓝牙耳机）。此 API 相当新，可能无法在所有浏览器和操作系统上可靠工作
