# 播放器提供者设置 <img src="/assets/icons/settings-players-icon.png" alt="预览图片" style="width: 70px; float: right;"  loading="lazy" />

所有播放器提供者都有可以配置的设置。设置页面看起来类似于以下内容。右上角的问号图标是指向本文档中相关页面的超链接。设置或可配置字段的特定帮助通常可用，通过选择此图标 !<a href="/assets/icons/question-mark.png" target="_blank"><img src="/assets/icons/question-mark.png" alt="image" loading="lazy" style="max-width: 100%;" /></a> 访问。所有提供者都可以被赋予自定义名称，并在必要时禁用。可以为所有提供者调整日志级别，这可以在高级设置中找到。（除非开发人员要求，否则不要选择 Verbose，因为它可能会产生不利的性能影响）。有关任何提供者特定设置（许多没有），请参阅提供者页面和[单个播放器设置](/music-assistant/settings/individual-player/)页面。
