# 音乐提供者设置 <img src="/assets/icons/settings-providers-icon.png" alt="预览图片" style="width: 70px; float: right;" loading="lazy" />

所有[音乐提供者](/music-assistant/music-providers/)都有可配置的设置。设置页面看起来类似于以下内容。文本`查看文档`是指向本文档中相关页面的超链接。通常可以为设置或可配置字段提供特定帮助，通过选择此图标 !<a href="/assets/icons/question-mark.png" target="_blank"><img src="/assets/icons/question-mark.png" alt="问号" loading="lazy" style="max-width: 100%;" /></a> 访问。所有提供者都可以被赋予自定义名称，并在必要时禁用。可以为所有提供者调整日志级别，这可以在高级设置中找到。（除非开发者要求，否则不要选择详细，因为它可能会对性能产生不利影响）。有关提供者特定设置，请参阅提供者页面。
