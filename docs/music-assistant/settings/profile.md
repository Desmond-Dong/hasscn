# MA 个人资料设置 <img src="/assets/icons/profile-icon.png" alt="预览图片" style="width: 70px; float: right;"  loading="lazy" />

此页面与当前登录的用户有关。用户名、显示名称和密码可以在此更改。

## 活动会话

- 显示已使用此用户登录的活动会话列表，可以使用垃圾桶图标注销

## 长期访问令牌

可以在此创建令牌，用于外部应用程序或允许从[未登录的设备](/music-assistant/faq/how-to/#access-the-ma-views-directly-via-url)访问前端。
