# 远程访问设置 <img src="/assets/icons/remote-access-icon.png" alt="预览图片" style="width: 70px; float: right;" loading="lazy" />

远程访问允许您从世界任何地方安全地连接到您的 Music Assistant 服务器。

## 配置

- 将远程访问状态切换为开启
- 对于拥有 Nabu Casa 订阅的用户，确保 `WebRTC 连接`选项（在 Home Assistant 云设置中）已开启

> [!NOTE]
> 除非在复杂的网络环境中运行，否则此功能不需要 Nabu Casa 订阅。例如，双 NAT、移动运营商或企业网络阻止标准 <a href="https://medium.com/@jamesbordane57/what-is-a-stun-server-df3563dbf14a" target="_blank" rel="noopener noreferrer">STUN 服务器</a>。这就是 TURN 服务器（Nabu Casa 提供）有用的地方，因为它们基本上中继流量。

设置是自动的，更多说明显示在 MA UI 中
