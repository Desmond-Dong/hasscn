# 用户管理 <img src="/assets/icons/user-management-icon.png" alt="预览图片" style="width: 70px; float: right;"  loading="lazy" />

此页面允许管理用户、他们的角色和权限。

## 管理用户

每个用户名右侧都有一个 ⋮ 菜单，其中有四个菜单项：
- <b>编辑用户。</b> 见下文部分
- <b>管理访问令牌。</b> 允许管理登录会话和长期令牌，与个人[个人资料设置](/music-assistant/settings/profile/)中显示的相同
- <b>禁用用户。</b> 阻止用户访问。设置被保留
- <b>删除用户。</b> 完全删除用户及其所有设置

## 编辑用户

!<a href="/assets/screenshots/edit-user.png" target="_blank"><img src="/assets/screenshots/edit-user.png" alt="预览图片" loading="lazy" style="max-width: 100%;" /></a>

单击用户名或通过 ⋮ 菜单时会出现上面显示的对话框。

每个用户可以被限制为一组特定的播放器。此外，将用户限制为某些音乐提供者有两个好处：在启动播放时仅使用这些提供者，并且收听统计数据准确地归因于用户的源流媒体账户。

### 多用户环境中的有声读物和播客进度同步

<a name="filter-progress-multi-user"></a>
某些播客和或有声读物提供者允许同步媒体项目的进度。通过音乐提供者筛选功能仅将此类提供者分配给特定用户，进度仅与此个人用户同步，而不是所有用户。通过多次设置这些音乐提供者，可以通过筛选功能正确分配提供者来实现个人用户同步。
