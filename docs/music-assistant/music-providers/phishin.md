# Phish.in 提供者 <img src="/assets/icons/phish-logo.png" alt="预览图片" style="width: 70px; float: right;"  loading="lazy" />

Music Assistant 支持 <a href="https://phish.in/" target="_blank" rel="noopener noreferrer">Phish.in</a>。此组件由 <a href="https://github.com/ozgav" target="_blank" rel="noopener noreferrer">Gavin</a> 贡献和维护。

Phish.in 为粉丝提供了一个全面的平台，可以发现、流式传输和分享乐队 Phish 的现场观众录音，该乐队以其即兴摇滚音乐而闻名。

## 功能

|           |                     |
|:-----------------------|:---------------------:|
| 免费订阅 | 是 |
| 自托管本地媒体 | 否 |
| 支持的媒体类型 | 艺术家、专辑、曲目、播放列表 |
| 支持[推荐](/music-assistant/ui/#view-home) | 否 |
| 支持歌词 | 否 |
| [电台模式](/music-assistant/ui/#track-menu) | 否 |
| 最高流媒体质量 | MP3 (16 bit, 44.1 kHz) |
| 登录方式 | 无 |

### 其他

- 流式传输从 1983 年至今所有有可用音频的 Phish 音乐会
- 浏览 1,800+ 场演出，在 Music Assistant 中组织为专辑
- 访问 50,000+ 个单独曲目表演
- 通过部分标题搜索音乐会、曲目和播放列表

## 配置：
- 在配置中，您只需单击保存按钮。

## 已知问题 / 说明

- 一些早期演出的音频不完整或缺失
- 媒体库视图有 2500 个项目限制，以保持 UI 响应性
- 搜索仅匹配标题（无模糊搜索或高级查询）
- 播放列表只读。访问 Phish.in 社区创建的用户播放列表
- 按年份、场地、标签、历史上的今天、最受欢迎的演出和曲目、用户播放列表浏览目录，并发现随机表演
