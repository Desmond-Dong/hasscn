# Podcast RSS Feed 提供者 <img src="/assets/icons/rss-icon.png" alt="预览图片" style="width: 70px; float: right;"  loading="lazy" />

Music Assistant 支持从 RSS 源流式传输播客。由 <a href="https://github.com/saeugetier" target="_blank" rel="noopener noreferrer">saeugetier</a> 贡献和维护。

Podcast RSS 提供者允许订阅播客源。RSS 源只提供单个播客。不支持聚合源。对于多个播客，必须配置多个提供者，每个都有唯一的源。

## 功能

|           |                     |
|:-----------------------|:---------------------:|
| 免费订阅 | 是 |
| 自托管本地媒体 | 否 |
| 支持的媒体类型 | 播客 |
| 支持[推荐](/music-assistant/ui/#view-home) | 否 |
| 支持歌词 | 否 |
| [电台模式](/music-assistant/ui/#track-menu) | 否 |
| 最高流媒体质量 | 有损，可变质量 |
| 登录方式 | 无 |

### 其他

- 允许解析通过配置 URL 检索的标准化 RSS 播客源

## 配置

在通用设置中添加完整的 RSS 源 URL。

## 已知问题 / 说明

- 只支持严格遵循标准化播客源格式的源
- 不会检查两个或更多提供者是否配置了相同的源
