# 内置提供者  <img src="/assets/icon.png" alt="预览图片" style="width: 70px; float: right;"  loading="lazy" />

内置提供者支持手动将曲目和电台 URL 添加到数据库，它还生成各种[播放列表](/music-assistant/usage/#playlists)。

## 功能

- 您可以添加可通过 URL 访问的曲目或任何在线电台。您也可以添加本地创建的电台，例如使用 <a href="https://icecast.org/" target="_blank" rel="noopener noreferrer">Icecast</a>。

## 设置

对于每个内置播放列表，都有一个开关可以启用或禁用它们。

## 使用

1. 在主菜单中选择"曲目"或"电台"
2. 然后单击右上角的图标

    !<a href="/assets/screenshots/url.png" target="_blank"><img src="/assets/screenshots/url.png" alt="截图" loading="lazy" style="max-width: 100%;" /></a>

3. 添加完整的 URL（包括 http:// 或 https://）以及可选的自定义名称和图片 URL。

完成第 3 步后，曲目或电台将在相应视图中可用。
