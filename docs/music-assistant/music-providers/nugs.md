# Nugs.net 提供者 <img src="/assets/icons/nugsnet-logo.png" alt="预览图片" style="width: 70px; float: right;"  loading="lazy" />

Music Assistant 支持从 <a href="https://www.nugs.net" target="_blank" rel="noopener noreferrer">Nugsnet</a> 流式传输您的音乐。由 <a href="https://github.com/brian10048" target="_blank" rel="noopener noreferrer">Brian</a> 贡献和维护。

> [!NOTE]
> 使用此提供者需要付费订阅
    
## 功能

|           |                     |
|:-----------------------|:---------------------:|
| 免费订阅 | 否 |
| 自托管本地媒体   | 否 |
| 支持的媒体类型 | 艺术家、专辑、播放列表 |
| 支持推荐 | 否 |
| 支持歌词 | 否 |
| <a href="https://www.music-assistant.io/ui/#track-menu" target="_blank" rel="noopener noreferrer">电台模式</a> | 否 |
| 最高流媒体质量 | 无损 FLAC (192 kHz / 24 bit) |
| 登录方式 | 密码 |

### 其他

## 配置

要配置，请输入用户名和密码。

## 已知问题 / 说明

Nugs 不支持收藏曲目。同样，没有可用的端点来传递 nugs 曲目 ID 并接收相关详细信息，如曲目名称或艺术家。

## 尚不支持

- Nugs.net 有电台流，但不可用
- 搜索
