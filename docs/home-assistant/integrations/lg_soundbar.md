**LG Soundbars** 集成可让您在 Home Assistant 中控制 [LG 条形音箱](https://www.lg.com/us/sound-bars)。

支持的设备：

- 配备以太网和/或 Wi-Fi 的 SK 与 SN 系列设备


:::note 配置
此集成可通过 UI 配置。前往 **设置 > 设备与服务** 添加。
:::

## 支持的操作

- 这些设备支持静音、音量控制、输入源选择和模式选择。
