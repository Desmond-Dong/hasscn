Ultraloq 智能门锁让用户无需钥匙即可进入家中。您可以通过 Ultraloq 应用或 Home Assistant 远程控制门锁。其母公司 U-tec 是 Works with Home Assistant 合作伙伴计划的成员，这意味着他们致力于确保 Ultraloq Z-Wave 产品保持最新状态，并可直接配合 Home Assistant 使用。

Ultraloq Z-Wave 系列智能门锁可在本地运行，并可与 Home Assistant 的 **Z-Wave** 集成无缝配合（需要 Z-Wave USB 棒或控制器）。

要添加 Ultraloq Z-Wave 产品，请将其作为 Z-Wave 设备进行配对：

[![Open **Add device** in your Home Assistant instance.](https://my.home-assistant.io/badges/add_zwave_device.svg)](https://my.home-assistant.io/redirect/add_zwave_device/?domain=ultraloq)

[进一步了解 Home Assistant 中的 Z-Wave。](/home-assistant/integrations/zwave_js/)
