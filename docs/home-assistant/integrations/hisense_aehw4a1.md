# Hisense AEH-W4A1

Hisense AEH-W4A1 是一款 Wi-Fi 模块，可为部分海信空调及其贴牌机型（如 Smart Cool、Beko 等）提供 Wi-Fi 连接能力。

后续空调机型使用的是其他 Wi-Fi 模块（如 AEH-W4B1 和 AEH-W4E1），这些模块与此集成不兼容，或尚未经过兼容性测试。

强烈建议不要同时使用官方手机应用和此集成，以避免指令冲突导致的问题。


:::note 配置
此集成可通过 UI 配置。前往 **设置 > 设备与服务** 添加。
:::
