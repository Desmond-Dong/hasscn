# Govee lights local

使用本地 API 控制，将 [Govee](https://www.govee.com/) 灯具接入 Home Assistant。

若要在 Govee 设备上启用本地控制，请参阅[这里](https://app-h5.govee.com/user-manual/wlan-guide)的说明。


:::note 配置
此集成可通过 UI 配置。前往 **设置 > 设备与服务** 添加。
:::
