# Xiaomi Sensor

The **Xiaomi Sensor** integration allows you to get data from your [Xiaomi](https://www.mi.com/) sensors.

The requirement is that you have set up the [Xiaomi Gateway (Aqara)](/home-assistant/integrations/xiaomi_aqara/) integration.
