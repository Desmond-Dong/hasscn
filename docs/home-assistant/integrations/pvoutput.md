**PVOutput** 集成会读取由您的太阳能光伏 (PV) 系统上传到 [PVOutput](https://pvoutput.org/) 的信息。


:::note 配置
此集成可通过 UI 配置。前往 **设置 > 设备与服务** 添加。
:::
