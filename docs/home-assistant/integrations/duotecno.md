您可以使用 **Duotecno** 集成在 Home Assistant 中控制 [Duotecno](https://www.duotecno.be/) 节点。

Home Assistant 目前支持以下设备类型：

- Climate
- Cover
- Light
- Switch


:::note 配置
此集成可通过 UI 配置。前往 **设置 > 设备与服务** 添加。
:::

您需要提供 Smart Box 的连接信息。
