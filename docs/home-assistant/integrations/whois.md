# Whois

**Whois** 集成可让您每天对自己拥有的域名执行 WHOIS 查询。它会提供 `expiration_date`、`name_servers` 和 `registrar` 等信息。


:::note 配置
此集成可通过 UI 配置。前往 **设置 > 设备与服务** 添加。
:::
