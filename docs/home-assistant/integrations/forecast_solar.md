[Forecast.Solar](https://forecast.solar/) 服务根据历史平均值结合天气预报，为您的太阳能电池板系统提供太阳能发电预测。

此集成提供有关您的太阳能电池板将产生多少能源的估计预测，让您可以提前规划如何最有效地使用产生的能源。

作为自动化示例，您可以确定：

- 您是否会在下一小时产生足够的太阳能来运行洗衣机。
- 您有电动汽车：确定明天是否会产生足够的太阳能来给车辆充电；如果没有，则在夜间低电价时段充电（部分）。

## 前提条件

Forecast.Solar 依赖于 [EU 光伏地理信息系统](https://re.jrc.ec.europa.eu/pvg_tools/en/tools.html) 提供的数据，您的太阳能电池板必须位于该工具覆盖的位置。数据几乎覆盖全球。

要使用 Forecast.Solar 集成，它需要一些关于您的太阳能电池板系统的信息：**纬度**、**经度**、**倾斜角**、**方位角**和**总组件功率**。

它需要具体位置（由 **纬度** 和 **经度** 定义），默认从您的 Home Assistant 配置的"家庭"位置获取。

**倾斜角**（tilt）是您的太阳能面板的角度（以度为单位）；是您的太阳能电池板与地球表面之间的角度。值为 `0` 表示它们水平放置在地球表面上，而值为 `90` 表示您的电池板垂直竖立。

**方位角**（以 360° 尺度的度为单位）；是您的太阳能电池板正面朝向的方向。由于一个完整的圆是 360°，值为 `0` 朝北，`90` 朝东，`180` 朝南，`270` 朝西（或中间的任何值）。

**总组件功率**（峰值瓦特）；太阳能电池板系统中的每个太阳能电池板都有一个最大功率峰值产值。为了为您的系统提供匹配的估计，Forecast.Solar 需要知道您的系统可以产生的总最大功率峰值。将所有电池板的最大峰值功率（以瓦特为单位！）相加作为此值。


:::note 配置
此集成可通过 UI 配置。前往 **设置 > 设备与服务** 添加。
:::

如果您有多个具有不同属性的太阳能组件平面（例如，屋顶的不同侧面在不同的组串上或不同的建筑物上，具有不同的方向或倾斜角），您可以多次添加集成并相应地设置参数。然后，您可以使用[模板传感器](/home-assistant/integrations/template/)来组合数据，例如，将不同平面的产量相加为一个值，作为您规划的基础。

## 传感器

Forecast.Solar 集成主要提供可在您的自动化中使用的传感器。

- 估计能源产量 - 今天（kWh）
- 估计能源产量 - 今天剩余（kWh）
- 估计能源产量 - 明天（kWh）
- 估计能源产量 - 本小时（kWh）
- 估计能源产量 - 下小时（kWh）
- 估计功率产量 - 当前（瓦特）

它还提供传感器，告诉您峰值产量将在哪个日期和时间（今天和明天）：

- 最高功率峰值时间 - 今天
- 最高功率峰值时间 - 明天

还有一些默认禁用的不太常用的传感器。如果您想使用这些，请在用户界面中启用这些实体：

- 估计功率产量 - 下小时（瓦特）
- 估计功率产量 - 下 12 小时（瓦特）
- 估计功率产量 - 下 24 小时（瓦特）

## 使用您的 Forecast.Solar 账户

[Forecast.Solar](https://forecast.solar/) 公共计划可以免费使用，但使用的数据分辨率较有限，因此此集成可以使用的数据细节较少。

如果您喜欢 Forecast.Solar 服务，或对更频繁的数据更新（基于更高的数据分辨率）感兴趣，可以[注册他们的一个计划](https://doc.forecast.solar/doku.php?id=account_models#compare_plans)。

要在此集成中启用 API 密钥的使用，请前往 [**Settings** > **Devices & services**](https://my.home-assistant.io/redirect/integrations/)。选择 Forecast.Solar 集成，然后选择 **配置**。输入您账户的 API 密钥。

## 调整估计

可以调整和优化估计以更好地匹配您的太阳能设置。有许多因素可能导致估计略有偏差（但请不要忘记，它始终是基于例如天气和历史数据的预测——而不是实际产生的功率）。

幸运的是，有一些控制措施可以使它们更适合您的情况，例如，通过稍微调整方位角或倾斜角。如果您的电池板在早晨/傍晚有些阴影，您可以考虑稍微衰减结果。

**衰减**因子允许您在早晨和傍晚稍微调整和"衰减"您的太阳能预测结果，这是使结果不那么乐观并更符合您实际情况的好方法。有关衰减的更多详细信息可以在[这里](https://doc.forecast.solar/damping)找到。

**逆变器**大小可用于您逆变器的最大功率低于所有太阳能电池板的总功率（在"总组件功率"下输入）的情况。因此，预测会考虑到最大功率不能超过您的逆变器可以处理的范围，从而为您提供更现实的预测图表。

[在 Forecast.Solar 文档中阅读有关衰减因子的更多信息](https://doc.forecast.solar/doku.php?id=damping&s[]=damping)。

要调整您的 Forecast.Solar 集成实例的配置设置：

1. 浏览到您的 Home Assistant 实例。
2. 前往 **[设置 > 设备与服务](https://my.home-assistant.io/redirect/integrations/)**。
3. 如果配置了多个 {{ name }} 实例，请选择您要配置的实例。
4. 选择 **配置**。
