# Home Assistant Hardware

**Home Assistant Hardware** 集成提供以下设备之间共享的功能：[Home Assistant Connect ZBT-2](/home-assistant/integrations/homeassistant_connect_zbt2/)、[Home Assistant Connect ZBT-1](/home-assistant/integrations/homeassistant_sky_connect/)、[Home Assistant Green](/home-assistant/integrations/homeassistant_green/) 和 [Home Assistant Yellow](/home-assistant/integrations/homeassistant_yellow/)。

## Configuration

此集成不提供用户可配置项。
