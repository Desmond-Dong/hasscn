The **The Energy Detective TED5000** integration monitors electricity consumption/production by connecting to the [TED](https://www.theenergydetective.com/) gateway, itself connected to one or several Measuring Transmitting Units (MTU). The platform creates up to two sensors per MTU, one for Wattage the other for Voltage.

If you want to enable the ted5000 integration, add the following lines to your "`configuration.yaml`" file.
:::tip
更改配置后需要重启 Home Assistant。
:::

```yaml
# Example configuration.yaml entry
sensor:
  - platform: ted5000
    host: 192.168.1.100
```

```yaml
host:
  description: The IP address of your ted gateway.
  required: true
  type: string
port:
  description: The port of your ted gateway.
  required: false
  type: integer
  default: 80
name:
  description: Name of the ted gateway.
  required: false
  type: string
  default: ted
```

For each plugged MTU, using an index starting at 1, the platform creates 2 sensors:

```yaml
sensor.<name>_mtu<MTU id>_power
sensor.<name>_mtu<MTU id>_voltage
```
