[HELTUN](https://www.heltun.com/) 恒温器可帮助您轻松实现供暖系统自动化；其触控面板开关可控制照明和电动设备；高负载开关则可控制灯组、加热器或单独插座等高功率设备。

### Firmware updates

HELTUN 会定期发布包含新增功能的固件，这些固件可通过加密 OTA（空中升级）流程发送到设备。在 Home Assistant 中，您可以前往设备页面，使用“更新设备固件”选项上传适用于该设备的 HELTUN 固件。
