# 系统检测到不受信任的内容

## 问题

Home Assistant 检测到不受信任的代码。看起来有人/某物将未签名的镜像或代码注入了您的 Home Assistant 系统。

## 解决方案

请确保您的网络没有受到入侵者的攻击。我们建议重新安装您的安装，并确保只有授权人员可以访问您的实例。
