## 问题

Supervisor 需要对主机上的 Docker 运行时拥有特权访问权限，才能执行它需要执行的所有操作。

## 解决方案

如果您运行的是较旧版本的 Home Assistant OS，请在[配置](https://my.home-assistant.io/redirect/config/)面板中更新它。

如果这不是我们的 Home Assistant OS，您的操作系统可能已过时。尝试检查并安装更新，然后重启系统。如果这不起作用，您可能需要重新运行我们的[便捷安装脚本](https://github.com/home-assistant/supervised-installer)。
