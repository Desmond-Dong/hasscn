# 虚拟化镜像不正确

## 问题

您的 Home Assistant OS 安装似乎运行在使用不适用于虚拟化环境的磁盘镜像的虚拟化环境中。

Home Assistant OS 发布了支持在虚拟化系统上运行的特定镜像。这些镜像经过优化，可在虚拟环境中运行，例如已安装客户工具并启用了半虚拟化驱动程序。

## 解决方案

您需要使用支持在虚拟化系统上运行的镜像重新安装 Home Assistant OS，[请参阅此处的说明](/home-assistant/installation/alternative)。
