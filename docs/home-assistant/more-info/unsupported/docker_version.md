# Docker 版本

## 问题

Supervisor 所需的版本取决于其正常工作所需的功能。

当前最低支持的 Docker 版本为：`20.10.17`。

但是，功能集会随着时间变化和改进。因此，最低要求的版本可能会更改。当发生这种情况时，在我们发布需要您升级 Docker 的版本之前会进行通知。

## 解决方案

如果您运行的是较旧版本的 Home Assistant OS，请[更新](https://my.home-assistant.io/redirect/updates/)它。

如果这不是 Home Assistant OS，您需要在主机上手动更新 Docker。有关如何操作的说明，请查看官方 [Docker 文档](https://docs.Docker.com/engine/installation/debian/)。
