## 问题

您已开启忽略作业条件的设置。Home Assistant 有一个内置保护机制，用于检测系统是否按预期工作。忽略作业条件会禁用此保护。

如果系统未按预期工作，某些任务可能会导致系统崩溃。

## 解决方案

要解决此问题，您需要通过使用命令行界面（CLI）移除所有忽略的条件来重新启用保护机制：

```bash
ha jobs reset
```

移除所有条件后，重启 Supervisor。

```bash
ha supervisor restart
```
