# Supervisor 源修改

## 问题

我们无法支持未经测试且未经 Home Assistant 项目批准的源修改。
当然，您可以自行修改运行您的安装，但这将导致失去安装的官方支持状态。

## 解决方案

恢复您的源修改并重启系统或重新全新安装所有内容。
