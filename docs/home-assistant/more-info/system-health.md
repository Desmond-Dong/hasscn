# 系统健康

系统健康会显示与您的 Home Assistant 实例相关的信息，包括系统架构、操作系统及其版本、Home Assistant 安装类型和版本、Python 版本、前端版本，以及您拥有的[仪表盘](/home-assistant/dashboards/)数量等数据。

系统健康集成数据可以在[**设置** > **系统** > **修复**](https://my.home-assistant.io/redirect/repairs/) > 右上角三点菜单中的 **系统信息** 下查看。或使用下面的 My 链接：

<img class="no-shadow" src='/home-assistant/images/more-info/system_health.png' alt="显示系统健康对话框的截图">

## GitHub 问题

GitHub 问题经常需要系统健康中呈现的数据。
Home Assistant UI 提供了一种简单的方法来获取此信息。

在系统信息对话框的右下角点击复制按钮。
这将格式化卡片的内容并将其放置在剪贴板上，以便您
可以轻松地将其粘贴到 GitHub 问题中。

<img class="no-shadow" src='/home-assistant/images/more-info/system_health_github.png' alt="显示系统健康对话框中复制按钮的截图">
