# 版本发布

Home Assistant 的新版本会在每个月的第一个星期三发布。确切日期可在 [Home Assistant Developers](https://developers.home-assistant.io/) 网站的即将举行活动日历中查看。

您可以在[这里](/home-assistant/blog/categories/core/)找到我们历次版本及其发布公告博客文章的列表。

我们发布周期的最后一周主要专注于 Beta 测试。参与 Beta 的用户可以在 [beta 发布说明](https://rc.home-assistant.io/latest-release-notes/) 中查看变更日志，并在 [Home Assistant 的 Discord 服务器](/home-assistant/join-chat) 的 `#beta` 频道获取帮助。我们也鼓励测试者[在 GitHub 上报告问题](/home-assistant/help/#bugs-feature-requests-and-alike)。
