# distutils.errors.DistutilsOptionError

如果您在 Mac 上使用 Homebrew 安装 Python，那么 `distutils.errors.DistutilsOptionError: must supply either home or prefix/exec-prefix -- not both` 是一个已知问题。请按照[这些说明](https://github.com/Homebrew/brew/blob/master/docs/Homebrew-and-Python.md#note-on-pip-install-user)解决此问题。
