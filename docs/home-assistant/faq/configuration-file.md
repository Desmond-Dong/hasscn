# 为什么使用 YAML 作为配置文件？

为什么不使用 JSON 或 XML 作为[配置文件](/home-assistant/docs/configuration/)？因为 [YAML](/home-assistant/docs/configuration/yaml/) 可以手写，你不需要关心逗号或标签，而且它是 JSON 的超集。
