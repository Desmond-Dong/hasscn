title: 我的集成没有显示出来
description: 我的集成没有显示出来
---

当集成没有显示出来时，可能有多种原因。在尝试以下任何步骤之前，请务必先查看[日志](/home-assistant/integrations/logger/#viewing-logs)，确认是否存在与您要设置的集成相关的错误。

如果您的配置文件中有错误的条目，可以使用 CLI 脚本检查您的配置，每种安装类型在常见任务中都有各自的章节：

- [Home Assistant Operating System](/home-assistant/common-tasks/os/#configuration-check)
- [容器](/home-assistant/common-tasks/container/#configuration-check)
