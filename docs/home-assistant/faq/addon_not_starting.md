# 为什么我点击插件的启动按钮时会闪烁红光？

如果您正在寻找关于无法启动或安装的插件的更多信息，请前往 UI 中的 [**设置** > **系统** > **日志**](https://my.home-assistant.io/redirect/supervisor_logs/) 并检查日志。

此页面上的日志与在自定义 CLI 中使用 `su logs` 看到的日志相同。
