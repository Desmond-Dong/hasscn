# 前端

Home Assistant 的用户界面可以通过多种方式进行自定义。

## 自定义选项

- [图标](/home-assistant/docs/frontend/icons) - 使用 Material Design 图标

## 前端功能

Home Assistant 前端支持：
- 多种仪表盘卡片
- 自定义主题
- Lovelace 仪表盘编辑器
- 移动端优化界面
- 深色模式
