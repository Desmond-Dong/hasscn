# 集成单个设备能源使用

Home Assistant 可以集成单个设备的能源使用情况。这样您就可以看到单个设备对总能耗的影响。除了能源使用外，Home Assistant 还支持跟踪单个用水设备的使用情况，让您能够监控家中特定设备的用水量。

## 能源监控硬件

### 智能插座

智能插座位于设备和插座之间，测量流经设备的能源。

根据您在家中使用的协议，您可以使用基于 Zigbee、Z-Wave 或 Wi-Fi 的插座。

### 智能继电器

智能继电器安装在您的“普通”开关后面，使其变得智能化。它允许您通过 Home Assistant 和连接的按钮或开关来控制设备。

## 用水监控硬件

要跟踪单个用水设备，您可以使用：

- 具有设备级监控功能的智能水表。
- 测量流向特定家电水流的在线流量计。
- 可报告自身用水量的智能家电（如洗衣机和洗碗机）。

关于水表硬件和集成的更多信息，请参阅[用水文档](/home-assistant/docs/energy/water/)。

## 带有功率（W）传感器的设备

一些智能设备，如空调、锅炉等，可能会提供功率传感器，以瓦特为单位测量。你可以使用 [积分（Riemann sum integral）集成](/home-assistant/integrations/integration/#energy)来计算设备所使用的能源。然后你可以在能源仪表板中将能源传感器作为单个设备使用。如果功率传感器具有适当的属性，你可以直接添加它。有关设置实体以在 **能源** 仪表板中使用的信息，请参阅[能源常见问题](/home-assistant/docs/energy/faq/#troubleshooting-missing-entities)。

## 上游设备和层级结构

您可以通过将一个设备设置为另一个设备的“上游设备”来创建设备层级结构。这意味着您可以在能源配置中建立设备之间的父子关系。这适用于能源设备和水设备。

例如，假设您有一个断路器监控某条电路的总能耗，同时也单独跟踪连接到该电路的各个设备。

对于用水量，您可能有一个主水表，以及用于洗衣机或洗碗机等家电的独立水表。如果不设置设备层级结构，Home Assistant 可能会重复计算这些用量。通过设置层级结构，它就能理解这些关系，并准确显示单个设备的用量而不会重复计算。

设置上游设备关系：

1. 将能源或水消耗实体作为单个设备添加。
2. 然后，在配置其他单个设备时，您可以选择之前添加的单个实体作为它们的上游设备。

这种层级视图可帮助您了解哪些设备正在从哪些来源消耗能源或用水，并防止用量被重复计算。

:::important
要设置层级结构，你必须先将所有相关实体作为单个设备添加到能源仪表板中。只有在单个设备下已列出的设备才能被选作其他设备的"上游设备"。
:::
