# Home Assistant 自动化

Home Assistant 包含您家中设备和实体的各种信息。您可以在仪表盘中查看这些信息，也可以用它们来触发自动化。这正是自动化最有趣的地方。

Home Assistant 中的自动化可让您对发生的事情自动作出响应。您可以在日落时打开灯光，或者在接到电话时暂停音乐。

如果您刚开始接触自动化，我们建议先从自动化蓝图开始。它们是由社区制作的现成自动化，您只需完成配置即可使用。

### [了解自动化蓝图 &raquo;](/home-assistant/docs/automation/using_blueprints/)

当您已经熟悉蓝图并想继续深入时，就可以进入下一步了。不过，在开始创建自动化之前，建议先了解自动化基础知识。

### [了解自动化基础知识 &raquo;](/home-assistant/docs/automation/basics/)
