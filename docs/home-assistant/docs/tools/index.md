# 工具

Home Assistant 提供多种工具帮助您配置和调试系统。

## 开发者工具

- [开发者工具](/home-assistant/docs/tools/dev-tools) - 调用动作、检查状态等
- [快速搜索](/home-assistant/docs/tools/quick-search) - 快速导航和搜索

## 配置工具

- [检查配置](/home-assistant/docs/tools/check_config) - 验证配置文件

## 工具用途

开发者工具可以：

- 调用动作以测试集成
- 查看和修改实体状态
- 测试模板
- 检查事件
- 无需重启即可重新加载配置
