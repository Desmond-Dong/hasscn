# 为语音计划做出贡献

Home Assistant Voice 依赖多种技术、模型和数据来源，来处理各种语言中的唤醒词和命令。我们以及其他社区的项目，都依赖志愿者贡献时间、声音样本或语言知识来不断改进。

请注意，模型是基于大量数据训练的，因此你做出的贡献可能需要一段时间后才能看到效果。

### 向 Wake Word Collective 贡献样本

花一分钟时间，使用[你自己的样本](https://ohf-voice.github.io/wake-word-collective/)帮助我们改进样本数据库。

### 向 Piper 捐赠你的声音

要创建一个 Piper 声音，并不需要一大群志愿者。你只需要一位该语言的母语者来创建数据集，随后将其训练成 Piper 模型，并再次由该母语者进行验证。[查看训练指南](https://github.com/OHF-Voice/piper1-gpl/blob/main/docs/TRAINING.md)。

缺少你所使用的语言吗？如果你愿意帮忙，请给我们发邮件！为了方便有兴趣的人参与语言贡献，我们开放了邮箱地址 [voice@nabucasa.com](mailto:voice@nabucasa.com)。

### 其他通过样本参与贡献的方式

你也可以参与公共数据集项目。许多语音转文本项目都会使用 Common Voice 和 Librivox 来训练模型。

- [Contribute samples to Mozilla Commonvoice](https://commonvoice.mozilla.org/en)
- [Contribute samples to Librivox](https://librivox.org/)

### 成为活跃贡献者

你还可以通过以下几种方式积极参与语音计划的开发：
- [Contribute sentence intents in your language](https://开发者.home-assistant.io/docs/voice/intent-recognition/contributing/)
- [Become a language leader](https://开发者.home-assistant.io/docs/voice/language-leaders)
- [Contribute code to Assist](https://github.com/home-assistant/核心/tree/dev/homeassistant/components/assist_pipeline)
