# 扩展 Assist

完成[最佳实践](/home-assistant/voice_control/best_practices/)中的步骤后，你的基础设置就已经齐备，可以开始使用 Assist 了。本节提供了一些思路，帮助你扩展设置以支持更高级的用法。

## 前提条件

在任意可用设备上让 Assist [正常运行](/home-assistant/voice_control/)，并按照[最佳实践](/home-assistant/voice_control/best_practices/)完成配置。

## 扩展 Assist 设置的一些思路

1. 添加自定义语句或修改现有语句。请参阅这个[自定义语句教程](/home-assistant/voice_control/custom_sentences/)。
2. 使用 AI 为 Assist [创建个性](/home-assistant/voice_control/assist_create_open_ai_personality/)。
3. 为 Assist 自定义[自己的唤醒词](/home-assistant/voice_control/create_wake_word/) - 仅当你拥有自己的硬件设置（如 [ESP32-S3 BOX](/home-assistant/voice_control/s3_box_voice_assistant/) 或 [ATOM Echo](/home-assistant/voice_control/thirteen-usd-voice-remote/)）时可用。
4. 如果你想打造真正高度定制的方案，可以[制作自己的 Assist 设备](/home-assistant/voice_control/make_your_own/)。
