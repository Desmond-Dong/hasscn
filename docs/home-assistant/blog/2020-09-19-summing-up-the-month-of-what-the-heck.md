# 总结“到底是什么？！”这个月

<a href='https://community.home-assistant.io/c/what-the-heck/52'><img src='/home-assistant/images/blog/2020-08-18-the-month-of-what-the-heck/social.png' class='no-shadow'></a>

一个月前，我写了[一篇博客文章][kickoff]，开始了我们的第一个月
“什么鬼？！” （WTH）曾经！一次实验，没有任何期望，
因为我们以前从未做过这样的事情。 But WTH, why not. 😁

今天，我想我们实际上可以说：一切美好的事情都会结束。

这是令人惊奇的一个月！已创建多达 746 个主题
[WTH 论坛][论坛]，总共收集了超过 4,500 个帖子。的内容
all these 创建 topics are amazing! And apparently, you all agreed on that,
所有这些帖子都获得了惊人的 2,600 个赞。

论内容，真是太精彩了！那里有一些非常好的反馈。
烦恼让很多人想：“是啊！什么鬼？！”更重要的是，
开发者也插话了！

开发人员投入并开始解决和修复问题！眼下
在编写过程中，有多达 68 个 Pull 请求，其中包含代码更改以解决这些 WTH
已经做了。其中很多已经随 [0.115 生日
发布][发布].

## WTH 论坛和 Hacktoberfest 闭幕

正如所说，一切美好的事情都会结束。 WTH 论坛即将结束。
这意味着无法创建新的 WTH 主题，但所有内容都将保留
可见的。

Hacktoberfest 即将来临，该论坛中的所有当前数据
确实可以帮助驱动它。对于开发人员来说，这是一个很好的资源
在十月期间做出贡献的机会。所以，当我们关闭时
下来，我们希望更多的内容能够得到解决。

为什么不一直运行这个 WTH 呢？嗯，随着时间的推移，它会失去力量。这是
当比较开始和结束的速度时现在已经可见
当月的。

不过，让我们看看明年是否可以再做一次。或许，我们应该结合起来
2021 年 Hacktoberfest 吗？

WTH这个月你过得怎么样？我们应该再做一次吗？
