# Hacktoberfest 2020

<img src='/home-assistant/images/blog/2020-10-01-hacktoberfest-2020/social.png' class='no-shadow'>

十月已经开始，这意味着：[Hacktoberfest][hacktoberfest]！

Hacktoberfest 是为期一个月的全球开源庆典。一个事件
向所有人开放。无论您是开发人员、学习编码的学生，
文档员、设计师，你可以帮助推动开源项目的成长，
就像家庭助理一样。

通过参加 Hacktoberfest，并贡献四个 GitHub 拉取
请求，您将获得一件免费 T 恤，或者今年新推出的一件
种植的树。鼓励所有背景和技能水平的人完成
黑客啤酒节挑战。

与其他年份一样，Home Assistant 参加了 Hacktoberfest。

## 做什么工作？

那么，您决定为自己赢得一件免费的 Hacktoberfest T 恤了吗？伟大的！
我们收集了一堆我们希望得到的任务/suggestions/bugs
一些帮助。您可以通过过滤 /searching 来查找以下问题：
[`hacktoberfest` 标签][全部]。

- **[Hacktoberfest 核心问题][核心]**
  
例如，帮助我们迁移
[unittest 测试到 pytest 风格测试][testcase] 功能或伸出援手
使用尽可能多的参考文献
在[配置流程翻译][翻译]中。

- **[前端 Hacktoberfest 问题][前端]**
  
我们可以使用一些帮助[将所有 `paper-dialogs` 移动到 `ha-dialogs`][论文]。

- **[文档 Hacktoberfest 问题][文档]**

[查找并修复损坏的链接][链接]在我们的网站上或帮助完成
向每个集成添加[正确的 IoT 类][类]。

或者查看我们在[所有家庭助理项目][全部]中的所有 Hacktoberfest 问题。

此外，我们最近举办了[“到底是什么？！”月][wth-博客]。期间
这个月我们的社区报告了很多建议和烦恼。
虽然这些主题可能尚未被批准作为更改，但这可能是一个很好的选择
为社区做出贡献的灵感来源。查看
[WTH 社区论坛帖子][wth]。

## 但是，我不是开发者？！

如果您不是开发人员，一般不熟悉 git、GitHub 或开源，
文档可能是一个很好的入门方式。一个相对简单的方法
贡献，是通过查看您使用或正在使用的集成文档
熟悉，检查一切是否仍然是最新的并且没有
拼写/grammar 错误。

我们网站上的每个文档页面都有一个“在 GitHub 上编辑此页面”，
在右上角。使用该链接，您可以更改该页面上的文本
并提出改进建议。

在我们的社区论坛上，有一个关于其工​​作原理的良好分步指南：
[在 GitHub 上编辑文档并创建拉取请求][docs-how-to]。

那么，你还在等什么？报名参加 [Hacktoberfest][hacktoberfest]
网站并开始黑客攻击！如果您有任何疑问，请访问我们的
[Discord chat server](/home-assistant/join-chat)。我们有专门的开发者渠道
很高兴为您提供帮助。

黑客啤酒节快乐🎉
