# Home Assistant Android Companion 1.5.0

今天我们发布了 Home Assistant Android Companion 1.5.0。这是一次非常棒的发布，带来了不少实用新功能，也有新的协作者加入一起完善它！

这次发布特别感谢我们的 Android 负责人 [@JBassett]，他修复了大量 Bug，新增了状态栏跟随应用主题的支持，并修复了在网络中发现已有 Home Assistant 实例的问题。

新增的一个很酷功能是主屏幕小组件。你可以用它快速触发你喜欢的脚本、场景或 Home Assistant 中的任意服务。感谢 [@KBerstene] 完成初始实现，也感谢 [@JBassett] 打磨配置界面。

如果你想把 Home Assistant 应用跑在墙上的平板上，这里有个好消息！感谢 [@neopilou]，现在应用新增了全屏模式选项。

想充分发挥我们 Android 和 iOS 应用的能力，可以查看 [Home Assistant Companion 网站](https://companion.home-assistant.io/)。

祝你使用愉快！如果你喜欢这款应用，别忘了给个评分，这能帮助更多用户发现它。
