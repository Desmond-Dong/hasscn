# 现状与展望

<p class='img'>
  <img src='/home-assistant/images/blog/2018-11-state-of-the-union/share-the-love.jpg' alt='Photo of Paulus Schoutsen presenting the 状态 of the union.'>
</p>

上周三（11 月 14 日），我们在阿姆斯特丹举办了一场由 <a href='https://www.ing.nl'>ING</a> 主办的线下聚会。80 多位 Home Assistant 爱好者到场，听我（Home Assistant 创始人 Paulus Schoutsen）分享。我讲了 Home Assistant 是如何起步的、它为什么能成功（社区！），以及我们正在努力的方向。
