# Home Assistant Cast（预览版）

今天是我的生日。我33岁了。为了庆祝，我想分享一个我一直在开发的项目预览：Home Assistant Cast。这是 Home Assistant Lovelace 界面，但是专为支持 Cast 的设备设计的，比如 Chromecast 或带有屏幕的 Google Assistant 设备。

<p class='img'>
<img src='/home-assistant/images/blog/2019-07-home-assistant-cast-preview/home-assistant-cast.jpg' alt='Photo of a Google Nest Hub running the Home Assistant Cast interface.'>
运行 Home Assistant Cast 界面的 Google Nest Hub 照片。
</p>

它可以在任何支持 Cast 的设备上运行，如果可用，还允许用户使用触摸屏与界面进行交互。该界面基于 Lovelace 构建，因此它具有您现在所习惯的同样强大的功能和实时交互体验。

Chromecast 应用通常从浏览器或 Android/iOS 移动应用启动。但如果只有一种解决方案，那就不是 Home Assistant 了。Home Assistant 与 Chromecast 的集成已有5年多的历史。这意味着您将能够作为自动化的一部分启动 Home Assistant Cast。例如，当检测到前门有访客时，您可以在电视上显示信息。太棒了！
