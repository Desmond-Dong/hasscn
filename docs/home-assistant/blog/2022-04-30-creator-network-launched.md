# 介绍 Home Assistant 创作者网络！

大家好！今天我们宣布推出一项新计划：Home Assistant Creator Network！我们非常期待能与创作者展开更紧密的交流，确保每个人都能获得尽可能准确、完善的信息。我们希望投入资源到社区中，并支持那些持续帮助用户了解 Home Assistant 的努力。这也是 Home Assistant 用来回馈那些积极与用户互动的创作者的一种方式。

## 什么是创作者网络？

创作者网络将汇聚一批持续活跃于社区、稳定产出内容，并不断为 Home Assistant 积累内容档案的创作者。这些创作者会为 YouTube、Twitter、Instagram、TikTok 等平台制作内容，目标都是帮助大家更好地使用 Home Assistant。这些内容可以是教程、公正的产品评测、酷炫的项目分享，等等。

我们期待把这个网络发展成一个既能彼此协作，也能与 Home Assistant 开发者直接合作的创作者群体。我们实现这一目标的重要方式之一，就是每月与网络成员进行通话，讨论版本发布和即将到来的变化。

访问 [Home Assistant Creators Network](https://partner.home-assistant.io/creators/) 了解更多信息以及申请方式。
