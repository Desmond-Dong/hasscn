2017 年 1 月 21 日，Home Assistant [宣布](/home-assistant/blog/2017/01/21/home-assistant-governance/)我们已实施行为准则。我们希望通过这篇博客告知你：我们正在更新行为准则。

## 为什么需要行为准则？

我们不断壮大的社区由来自世界各地的人组成。我们希望确保每个人都能在社区中获得无骚扰的体验，不受年龄、体型、可见或不可见残障、族裔、性特征、性别认同与表达、经验水平、教育程度、社会经济地位、国籍、外貌、种族、宗教或性身份与性取向的影响。

行为准则说明了社区中哪些行为不可接受、我们将如何处理违规行为，以及你可以如何报告事件。

这能让我们的社区保持开放、友好、多元、包容和健康。

## 具体更新了什么？

我们的行为准则基于 [Contributor Covenant](https://www.contributor-covenant.org/) 制定。该准则已被全球超过 20 万个开源项目采用。现在它已更新到 2.0 版本，Home Assistant 也将同步采用该版本。

总体来说，与[我们原有版本](https://github.com/home-assistant/home-assistant.io/pull/13588/files)相比，变化并不大，主要是措辞优化。最重要的变化是新增了执行指南，这将帮助 Home Assistant 的成员、贡献者、版主、管理员和维护者更一致地执行准则。

从现在开始，这份更新后的行为准则将适用于 Home Assistant 组织下的所有项目、社交渠道和社区沟通渠道。
