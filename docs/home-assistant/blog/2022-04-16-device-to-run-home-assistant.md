title: Buying a 设备 to run Home Assistant (April 2022 edition)
description: Chip shortages and global supply chain issues make getting a 设备 complicated.
  Here is our latest overview.
---

Home Assistant 是一个专注于隐私和本地控制的家庭自动化平台。即使没有互联网连接，它也可以完整运行。Home Assistant 在日常使用中已经足够易用，而我们也一直投入大量时间持续优化这套体验。

由于全球芯片短缺和供应链问题，目前想买到一台能运行 Home Assistant 的设备非常困难。为了解决这个问题，我们推出了 [Home Assistant Yellow](/home-assistant/yellow/)，这是一台预装 Home Assistant 的设备。坏消息是：Home Assistant Yellow 目前还没有准备就绪，要到今年稍晚才会开始发货。

目前正有一大波用户开始关注如何运行 Home Assistant（[某种程度上要“感谢” Insteon](https://staceyoniot.com/insteon-is-down-and-may-not-be-coming-back/)），所以我觉得现在正是一个合适的时间，来盘点截至 2022 年 4 月可以买到 Home Assistant 运行设备的最佳选择。

Home Assistant 有[多种安装形式](/home-assistant/installation/#compare-安装-methods)。如果你是新用户，我们建议使用我们的操作系统版本。它正是 Home Assistant Yellow 的基础，也能把设备变成一个用户友好、易于维护的家庭自动化中枢。

**目前最值得购买且仍有现货的 Home Assistant 运行设备，是 Ameridroid 提供的 [ODROID-N2+ Home Assistant Bundle](https://ameridroid.com/collections/kits/products/odroid-n2-home-assistant-blue-bundle-limited-edition)。** 这套组合包含 ODROID-N2+，它也是我们 [Home Assistant Blue](/home-assistant/blue/) 所采用的硬件平台。它预装了 Home Assistant，到手即可使用，而且今天就能买到。

ODROID-N2+ 也可以从其他商店或直接从制造商 [Hardkernel](https://www.hardkernel.com/shop/odroid-n2-with-4gbyte-ram-2/) 购买。不过这些版本不会预装 Home Assistant，需要你按照 [Home Assistant 安装指南](/home-assistant/installation/odroid) 自行完成安装。

如果你并不急着立刻开始使用 Home Assistant，我们推荐选择 [Home Assistant Yellow](/home-assistant/yellow/)。这是我们自己打造的理想 Home Assistant 硬件方案，支持 Zigbee、未来也将支持 Matter，并且配备 M.2 插槽，拥有更好的长期扩展性。

到目前为止，大家运行 Home Assistant 最受欢迎的方式仍然是在树莓派 4 上完成。由于供应链问题，树莓派目前非常短缺，每次补货后都会很快售罄。想买到它，最好的办法是使用 [rpilocator.com](https://rpilocator.com/)，这个网站会追踪所有官方授权经销商的树莓派库存情况。

如果你在 Twitter 上关注他们，可以通过点击其个人主页上的铃铛图标，在他们发布新的到货提醒时收到通知。顺便提醒一下，如果你打算从 Adafruit 购买树莓派，需要先为账户启用双因素认证（这是为了防止机器人抢购）。
