TL;DR：Ubiquiti Networks 已聘用 Home Assistant 创始人 Paulus Schoutsen，支持 Home Assistant 继续实现这些目标：让用户配置更简单、与设备厂商集成更深入，并让集成的创建、维护与演进更轻松。

Home Assistant 是一个开源项目，到目前为止主要靠大家在业余时间推动。过去四年半里，它从我一个人维护、只有少量集成的小框架，成长为拥有自己的操作系统、超过 1000 个集成、优秀性能、900 多位贡献者，并且主 Docker 镜像下载量超过一千万次的项目！

Ubiquiti Networks 看到这样的增长和充满热情的社区后，主动联系了我们。

Ubiquiti Networks 目前专注于三大技术方向：高容量分布式网络接入、统一信息技术平台，以及面向家庭与个人场景的新一代消费电子。企业级品质加上亲民价格，让他们在我们的用户中非常受欢迎。他们也和我们一样重视“尽量避免云依赖”。例如他们的 [UniFi Video] IP 监控方案，就是完全本地托管。

他们看到了 Home Assistant 成为家庭自动化事实标准平台的潜力：快速、开源、本地。他们也希望进一步深化 Ubiquiti 产品与 Home Assistant 的集成，甚至未来支持在其硬件上运行 Home Assistant 实例。

因此，我们达成一致：我（Paulus，Home Assistant 创始人）将以全职员工身份加入 Ubiquiti Networks，专注推动 Home Assistant 发展。这样我就能把全部精力投入到让用户配置更简单、加强与设备厂商的集成，以及让集成更易创建、维护和演进。

Ubiquiti Networks 不会获得 Home Assistant 的所有权。我们仍将保持独立与开源，只是在 Ubiquiti Networks 的支持下，以前所未有的速度持续改进。

我对这个机会非常激动，2018 会是 Home Assistant 特别精彩的一年！
