---

早在 2021 年，我们发起了 Home Assistant Yellow 的众筹活动。Home Assistant Yellow 体现了我们心目中智能家居中枢应有的样子，包含了爱好者想要的一切功能。我们对 Home Assistant Yellow 非常自豪，但与此同时，我们也越来越清楚地意识到，刚刚开始智能家居之旅的人，并不需要 Yellow 提供的全部功能。再加上全球元器件短缺让所需的 Compute Module 4 更贵也更难买到，Home Assistant Yellow 已经不再适合新手入门。

我们认真对待了这一点，因此今天，我们为新用户推出了一种经济实惠的方式，把他们的智能家居升级到 Home Assistant。它就是 Home Assistant Green，售价仅为 99 美元（建议零售价）。

![Home Assistant Green 照片](/home-assistant/images/blog/2023-09-ha10/home-assistant-green.jpeg)

<!--more-->
Home Assistant Green 在一个盒子里提供了运行 Home Assistant 所需的一切。只要把随附的电源线和网线接到这台中枢上，你就可以开始使用。随后，Home Assistant Green 会自动安装最新版本的 Home Assistant，而应用程序（移动端或网页端）会引导你完成设置流程，并立即发现你的智能设备。这是开始使用 Home Assistant 最简单的方式，也非常适合你自己，或者想迈出家庭自动化第一步的亲友。

<p class='img'>
<img src="/home-assistant/images/blog/2023-09-ha10/green-with-yellow-and-blue.jpeg">
Home Assistant Green 与 Home Assistant Yellow、Home Assistant Blue 同框
</p>

在内部，Home Assistant Green 搭载了一颗强大的 1.8&nbsp;GHz 四核 CPU。它配备了充足的内存与存储空间，提供 4&nbsp;GB RAM 和 32&nbsp;GB eMMC，完全足以让你在这台设备上长期运行 Home Assistant。连接你现有的智能家居平台后，你就能立刻使用通过 Zigbee、Matter 或其他智能标准通信的设备。当你的智能家居进一步扩展时，你可以使用 <a href="/home-assistant/skyconnect">Home Assistant SkyConnect</a> 来增加 Zigbee 和 Thread 无线电（Thread 支持仍在开发中），或者使用许多第三方 USB 设备之一，为 Home Assistant Green 扩展像 Z-Wave 或蓝牙这样的其他智能标准支持。

在外观上，Home Assistant Green 采用了惊艳的半透明外壳设计，让你能看到中枢内部的硬件。与某些其他智能家居设备不同，它在你的家居环境中看起来也很漂亮。外壳底部同时兼作这台中枢的散热器，由整块黑色阳极氧化铝制成。当然，这个外壳顶部还带有[全新的 Home Assistant 标志][logo-博客]，以低调的马赛克图案呈现。
