今天我们正式发布全新的[数据科学门户][post]，帮助你从自己的智能家居数据中获得洞察。你可以了解 Home Assistant 在本地存储了哪些数据，也能学习如何探索这些数据。只需 15 分钟，你就能搭建本地数据科学环境，并开始对自己的数据生成报告。

我们为流畅体验做了很多工作。基于 Jupyter 的本地数据科学环境可以一键搭建，并且预装了 Home Assistant 入门 notebook，能自动连接你的数据库并一键生成报告。

这一切由 [@frenck] 提供的全新 JupyterLab Hass.io 插件，以及 [@robmarkcole] 提供的全新 HASS Data Detective Python 包共同驱动。

今天只是开始。我们会持续为 Home Assistant 打造数据工具。我们的目标是让 Home Assistant 成为家庭自动化数据科学从业者最易用的平台。
