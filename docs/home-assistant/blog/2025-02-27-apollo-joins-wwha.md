[Apollo automation](https://apolloautomation.com/) 已加入 [Works with Home Assistant](https://partner.home-assistant.io/) 计划。他们为三款最新设备完成了认证，其中包括社区最受欢迎的存在感和空气质量多传感器，让用户在 Home Assistant 中使用这些设备时获得尽可能出色的体验。

值得一提的是，这是首批获得 Works with Home Assistant 认证的 [ESPHome](https://esphome.io/index.html) 设备。Apollo 也是一个重要趋势中的先行者，而我们对此非常兴奋：Home Assistant 社区成员正在把他们的开放项目发展成可持续的业务，为繁荣的 Open Home 生态系统提供支持。

<!--more-->

## 背景

Apollo 自动化是一家不断成长的家族企业，深深扎根于 Home Assistant 和 ESPHome 社区。没错，他们的名字确实来自他们的德国牧羊犬 Apollo 🐕。

<div class="alert">
    <p>Apollo 自动化由 Home Assistant 社区打造，也为 Home Assistant 社区而生。我们相信开放的智能家居技术应当赋予用户控制权、隐私保护和灵活性。加入 Works With Home Assistant 计划对我们来说是自然而然的一步，它能确保我们的设备无缝集成，同时也支持我们热爱的生态系统。Home Assistant 让用户能够打造强大的智能家居，而我们也很高兴通过制造高质量、可靠的硬件，为所有人带来更好的体验。</p>

两年前，他们开始与社区分享一款体积极小、由 ESPHome 驱动的新型多传感器设计。通过收集反馈、共享迭代过程以及协作开发，他们把这款传感器变成了真正的产品，而且它天生就是为 Home Assistant 而打造的。他们一直保持这种开放、社区优先的方式，现在已经打造出多款更加成熟精致的多传感器产品。

给蚂蚁准备的多传感器？🐜</p>

Home Assistant 是[唯一真正的智能家居平台](https://newsletter.openhomefoundation.org/true-platforms-let-you-create/)，因为它让每个人都能完全掌控并自由构建。 [ESPHome](/home-assistant/integrations/esphome/) 则是智能设备领域的终极平台，它正在孕育一个全新的 Open Home 项目生态。ESPHome 完全本地运行，是改装爱好者的天堂，让用户能够自己动手构建智能家居设备（可以看看 [ESPHome 入门指南](https://esphome.io/guides/getting_started_hassio.html)）。

创作者也可以用它来打造即插即用的成品设备，为用户带来非常顺畅的体验。设备可以被发现、轻松加入你的 Wi-Fi 网络和 Home Assistant，同时还能在 Home Assistant 中一键更新。

ESPHome 和 Home Assistant 让任何人都可以把自己热爱动手创造的兴趣变成职业，而他们打造出的产品又反过来支持更有活力的 Open Home。Apollo 是 Works with Home Assistant 计划中一个令人振奋的新伙伴，也应当成为社区所有创客的灵感来源。

## 设备

Works with Home Assistant 与其他认证计划不同，它会在内部进行严格测试，确保产品开箱即可与 Home Assistant 无缝协作。加入计划的公司还承诺提供长期支持和固件更新，并在 Home Assistant 社区中发挥积极作用。Works with Home Assistant 由 [Open Home Foundation](https://www.openhomefoundation.org/) 运营，而 [Home Assistant Cloud](/home-assistant/cloud/) 订阅者的支持正为这项工作提供资金。

Apollo 首先选择为以下三款产品进行认证：

- [MTR-1 Multi-Target Radar Multisensor For Home Assistant (LD2450)](https://apolloautomation.com/products/mtr-1) - 小巧但强大的毫米波存在感传感器，还可以集成多种其他传感器。
- [MSR-2 mmWave Multisensor For Home Assistant (LD2410B)](https://apolloautomation.com/products/msr-2) - 更小巧的多传感器，是最初那个社区共创设计的继任者。
- [AIR-1 Air Quality 传感器 For Home Assistant](https://apolloautomation.com/products/air-1) - 一款小巧的空气质量多传感器，还可以扩展接入大量不同类型的传感器。

我们预计 Apollo 还会把更多产品线带入该计划，也很期待他们接下来的新创新！这只是今年众多 Works with Home Assistant 公告的开始，请继续关注！
