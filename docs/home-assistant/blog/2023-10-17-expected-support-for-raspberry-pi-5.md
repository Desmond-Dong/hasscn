---
title: 预计 Home Assistant OS 将支持 Raspberry Pi 5
description: 预计在 2023 年底或 2024 年初为 Raspberry Pi 5 提供稳定的 Home Assistant OS
  支持
---

9 月 28 日，Raspberry Pi 发布了 [Raspberry Pi 5](https://www.raspberrypi.com/news/introducing-raspberry-pi-5/)，让全世界都感到惊喜，说实话，我们也是如此。这块新主板承诺将提供超过 Raspberry Pi 4 两倍的性能，目前已经开放预购。Raspberry Pi 预计会在 10 月底前向客户发货。

根据我们的[分析数据](https://analytics.home-assistant.io/)，目前有三分之一的 Home Assistant 用户将 Raspberry Pi 4 作为专用 Home Assistant 系统。事实上，Home Assistant OS 也是 Raspberry Pi 设备上[安装量第三高的操作系统](https://rpi-imager-stats.raspberrypi.com/)。因此，我们相信很多人都在期待用全新的 Raspberry Pi 5 升级自己的 Home Assistant 安装。我们和你一样期待这次新发布，并且一旦收到 Raspberry Pi 提供的预发布开发板，就会立即开始适配开发！

由于我们并未参与 Raspberry Pi 的 Beta 计划，因此为 Home Assistant OS 添加 Raspberry Pi 5 支持的工作尚未开始。现阶段仍然很难准确估计这项工作的规模，但我们想强调，这是一项重要任务，我们希望把它做好。虽然 Beta 版本会更早推出，但我们目前预计稳定版要到今年年底或 2024 年初才会发布。

这意味着在 Raspberry Pi 5 首发时，你还无法在其上运行 Home Assistant OS。虽然也有[其他不使用 Home Assistant OS 的安装方式](/home-assistant/installation/)，但我们只建议高级用户采用。如果你当前正在使用 Home Assistant OS，并且已经预购了 Raspberry Pi 5，我们建议等到 Raspberry Pi 5 的稳定版支持发布后，再迁移你的安装。

对于 Home Assistant Yellow 用户，Raspberry Pi 目前还没有就基于 Raspberry Pi 5 的 Compute Module 5 发表任何说明。只有当他们公布新 Compute Module 及其规格信息后，我们才能判断它是否兼容 Home Assistant Yellow。我们也想指出，Raspberry Pi 4 发布到 Compute Module 4 发布之间相隔了 16 个月。对于目前仍在寻找 Compute Module 4 来完成 Home Assistant Yellow 套件的用户，我们也很高兴地告诉你，根据 [rpilocator](https://rpilocator.com/?cat=CM4) 的信息，Compute Module 4 现在已经再次更容易买到了。

**更新（10 月 19 日）**：我们已在本周早些时候收到了 Raspberry Pi 5 开发板，目前正在评估 Home Assistant OS 的支持方案。感谢 Raspberry Pi 团队！如果你对 Raspberry Pi 5 支持的技术讨论感兴趣，或者只是想跟进进展，我们已经在 GitHub Discussions 上创建了一个[专门讨论 Raspberry Pi 5 的主题帖](https://github.com/home-assistant/operating-system/discussions/2844)。
