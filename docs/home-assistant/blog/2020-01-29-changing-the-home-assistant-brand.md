---
title: 调整 Home Assistant 品牌命名
description: 我们正在调整 Home Assistant 的品牌命名，让概念更清晰、沟通更简单！
---

过去几年里，我们在 Home Assistant 体系下逐步积累了很多项目。我们最初只是一个 Python 应用（名为 Home Assistant），随后又加入了操作系统（HassOS）和管理系统（Hass.io Supervisor），让用户可以更轻松地管理家庭自动化，同时也能更容易保持系统更新。

这些不同项目最终都汇聚在 Hass.io 中。它是一个安全、易于更新，并运行着全球最强大家庭自动化平台的系统。但 Hass.io 与 Home Assistant 并存的命名方式，确实让很多人感到困惑，尤其是刚加入我们社区的新用户。

让 Home Assistant 更简单，下一步就是让命名更简单。我们的目标是让私有家庭自动化对每个人都触手可及，而这其中的重要一环，就是让大家知道该如何准确称呼这些东西。

Home Assistant 是面向家庭的自动化操作系统。
为了让这句话更清楚，我们将把 Hass.io 更名为 Home Assistant，
并将原本的 Home Assistant（Python 应用）更名为 Home Assistant Core。

**注意：** 这次变更仅涉及项目命名，不会影响任何项目功能。

如果你目前是在 Docker 容器中运行 Home Assistant，或是在 Python 虚拟环境中运行，那么你使用的就是 “Home Assistant Core”。
Home Assistant Core 会一直保持为一个可独立运行的应用，这一点不会改变。
我们保证。

这次命名调整是件大事！不仅大家需要时间适应，涉及到的各处重命名工作量也非常大。完成主要部分的品牌替换，预计还需要几周时间。

所以这段时间请大家多包涵。
