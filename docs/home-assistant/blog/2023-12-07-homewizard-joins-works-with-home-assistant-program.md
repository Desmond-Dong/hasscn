# HomeWizard joins Works With Home Assistant program

我们很高兴宣布，HomeWizard 已加入 Works With Home Assistant 计划，并获得 *“Works Locally With Home Assistant”* 标识，同时承诺确保其产品在 Home Assistant 中拥有良好表现。这家来自荷兰的公司生产一系列 Wi-Fi 设备，帮助家庭了解自己的用电、用水和用气情况。他们希望让人们更关注自身的能源消耗，并帮助大家节省能源，这不仅对钱包有好处，也对世界更有益。这一使命与我们的[开放家庭价值观](/home-assistant/blog/2021/12/23/the-open-home/)高度契合。他们也将成为首家使用新版 Works With Home Assistant 标识的公司，新标识中采用了全新的 Home Assistant logo！

![HomeWizard works locally with Home Assistant](/home-assistant/images/blog/2023-12-homewizard/homewizard-wwha.png)

<!--more-->
HomeWizard 成为合作伙伴的历程，始于一位负责其能源电表产品的工程师。他在业余时间开发了一个自定义 Home Assistant 集成。这个自定义集成随后不断成长，并在 2022.2 版本中成为 Home Assistant Core 的一部分；随着它在 HomeWizard 用户中的人气不断上升，HomeWizard 也正式采用了该集成，并在今天成为官方合作伙伴！

因此，[HomeWizard Energy 集成](/home-assistant/integrations/homewizard/) 的开发过程始终在倾听 Home Assistant 社区的反馈。像产品内置本地 API、允许禁用云端通信等功能，都是在这些反馈基础上加入的。目前，该集成支持以下产品：

Wi-Fi P1 Meter（型号：HWE-P1）：提供电力输入/输出、能耗（单相或三相）、智能电表信息以及燃气相关的传感器。

Wi-Fi Energy Socket（型号：HWE-SKT）：提供电力输入/输出、能耗传感器，以及用于控制插座的开关实体。

Wi-Fi Watermeter（型号：HWE-WTR）：提供当前和累计用水量传感器。

Wi-Fi kWh Meter（型号：SDM230-WiFi、SDM630-WiFi）：提供电力输入/输出和能耗传感器。

虽然 HomeWizard 是一家荷兰公司，但它的产品可以在很多国家使用。例如，P1 Meter 适用于[大多数带有 P1 接口的现代电表](https://helpdesk.homewizard.com/en/articles/5935311-is-my-smart-meter-compatible)，Water meter 适用于[带有 Itron、Elster 或 Sensus 安装孔位的模拟水表](https://helpdesk.homewizard.com/en/articles/6287701-is-my-analog-watermeter-compatible)，而 kWh Meters 则可以安装在 DIN 导轨上。以上产品都可以通过 [HomeWizard 网店](https://www.homewizard.com/shop/) 购买。

我们对这一消息感到非常兴奋，也很喜欢 HomeWizard 集成从一位工程师的个人兴趣项目一路成长为官方合作伙伴的过程。你也可以查看[我们最初的公告](/home-assistant/blog/2022/07/12/partner-program/)，了解更多关于 Works with Home Assistant 合作伙伴计划的信息。
