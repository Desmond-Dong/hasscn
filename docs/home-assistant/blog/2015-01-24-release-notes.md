# 2015 年 1 月 24 日发布说明

我刚刚把开发分支的最新版本合并到了 master。以下是本次发布的一些亮点：

**通过前端完成配置**<br>
Philips Hue 现在可以被自动发现，并会使用新的 `configurator` 组件与用户交互来完成设置流程。

**Wink Hub 支持**<br>
感谢 John McLaughlin 和 Geoff Norton 的工作，我们现在支持连接到 Wink Hub 的灯光、开关和传感器。

**入门指南与组件页面已重组**<br>
入门说明现已拆分为“每个组件单独页面”加“总览页面”。目标是让每个组件页面都包含：

- 它的作用
- 如何配置
- 维护哪些状态
- 提供哪些服务
- 额外的开发提示

**更合理的错误提示**<br>
如果你的 Python 3 版本不正确、忘记克隆 git submodule，或未安装依赖，Home Assistant 现在会给出更清晰的错误提示和解决建议。

**首次启动流程更顺畅**<br>
Home Assistant 现在支持 `--open-ui` 和 `--demo-mode` 命令行参数，可自动打开浏览器并展示可用内容。现在也支持无密码运行，使界面启动后可以自动登录。
