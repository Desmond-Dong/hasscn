# 全新开发者网站

2014 年 12 月我们上线官网时，只有少量组件和使用说明。此后网站成长到超过 1000 个页面，覆盖了非常广泛的话题。增长固然可喜，但也给文档组织方式带来了很大压力。受影响最明显的就是开发者部分：它长期挂在顶部菜单的一个入口下，所有内容都挤在同一个侧边栏里。

为了解决这个问题，我们发布了新网站：[开发者.home-assistant.io](https://开发者.home-assistant.io)。这个站点面向 Home Assistant 开发者，提供如何搭建开发环境、修复 bug、参与翻译和改进前端等资源。主站将专注于服务 Home Assistant 用户。

欢迎访问并告诉我们你的看法：[开发者.home-assistant.io](https://开发者.home-assistant.io)。
