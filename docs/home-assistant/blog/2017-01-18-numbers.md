# Numbers

2017 年才来到第 3 周，但已经发生了很多很棒的事情。这里做一个简单回顾。

- 在 OSS Metrics 排行榜上，我们已经来到了第 30 名。仅仅三个月前，也就是 2016 年 9 月，我们的起点还是第 66 名。
- 我们登上了 [Github Trending](https://github.com/trending/python)。同时，[@balloob](https://github.com/balloob) 也被提名为热门开发者。
- [@balloob](https://github.com/balloob) 在 OpenIoT Summit 2016 上的演讲，被评为大会的 [Top 5 videos](https://www.linuxfoundation.org/博客/2017/01/top-5-videos-from-embedded-linux-conference-and-openiot-summit-2016/) 之一。
- 我们现在已经提供超过 [500](/home-assistant/integrations/#all) 个组件和平台。
- 到目前为止，我们已经在[主仓库](https://github.com/home-assistant/home-assistant)中处理了超过 3500 个 Pull Request。

你也许会问，这为什么这么了不起？因为 Home Assistant 完全是一个由志愿者推动的社区项目。我们没有资金支持，没有公司在背后运作，也没有任何受薪开发者专职开发 Home Assistant。所以，这里也想再次对你说一声“谢谢”，因为你正是推动 Home Assistant 前进的力量。

想看更多数字？去看看 [Trivia page](/home-assistant/help/trivia/#numbers) 吧……

- Fabian
