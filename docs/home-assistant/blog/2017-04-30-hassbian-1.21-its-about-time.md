### Hassbian 1.21 - It's about time isn't it
由于我这个 HASSbian 的开发者最近在搬家、换工作，等等，能投入到 HASSbian 开发上的时间并不多。1.21 版本其实已经预发布了几个月，只是一直没有很好地对外说明。希望这次发布能改变这种情况，我也会尽量更频繁地发布新版本。从 1.1 升级到 1.21 并没有简单的方法，不过其中 95% 的变更都可以通过安装 [hassbian-config][hassbian-config-发布] 软件包来完成。更多信息请参阅 [hassbian-config][hassbian-repo] 页面。

### Hassbian 1.22 - Sins of last night
开发有时会进展很快，也很令人兴奋，但偶尔也会犯错。

在发布 1.21 时，OpenZWave 构建脚本中的一个小问题没有被修正，尽管它已经是个已知 bug。这个问题并不复杂，只是 OpenZWave 配置文件夹的符号链接创建有问题。现在这个问题已经修复，我们也顺便加入了一些已经准备好的新内容。下面的列表已经补充了更新后的信息。

### Hassbian-config

为了让你可以进一步自定义安装，我们加入了一个名为 `hassbian-config` 的工具。这个工具附带了一组可以轻松安装的软件包，帮助你更方便地定制 Home Assistant 安装。这也取代了 1.1 中的 `hassbian-scripts` 功能。

- 安装 Hue。配置 Python 可执行文件，使其可以使用低编号端口，以便配合 Emulated Hue 集成与 Amazon Echo、Google Home 和 Mycroft.ai 一起使用。
- 安装 Mosquitto MQTT server。从 Mosquitto 项目的官方仓库安装最新的 Mosquitto 软件包和客户端工具。现在还包含了 websocket 支持。
- 安装 Libcec。添加本地 [HDMI-CEC support][cec]。
- 安装 Open Z-Wave-pip。通过 pip 软件包安装 Python Open Z-Wave。这是安装 Z-Wave 支持最快、也是推荐的方式，但不会预先安装 OZWCP。
- 安装 Open Z-Wave。从 git 安装 Python Open Z-Wave 和 OZWCP。
- 安装 Samba。允许网络中的任何人从任何计算机编辑你的配置。如果你与他人共享网络，由于这个共享没有安全保护，因此不建议使用。
- 安装 Tellstick。安装 Tellstick 软件包，以便控制和使用已连接的 Tellstick。
- 安装 Tradfri。安装使用 IKEA Trådfri 所需的依赖项。

### Spring cleaning

这个镜像还对基础系统以及用于生成树莓派镜像的脚本做了不少清理工作。

- Replaced the `hassbian-scripts` folder with `hassbian-config`.
- 更新 `pi-gen`。我们的构建脚本已经升级，再次更贴近 Raspbian 镜像。现在如果你愿意，也可以使用 Docker 来构建这个镜像。
- Added libtool and autoconf package. 这些是某些 pip 软件包所需的依赖。
- Pi ZeroW 现在应该可以运行这个镜像了。
