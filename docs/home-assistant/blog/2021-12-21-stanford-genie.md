# Demo: Genie, privacy-preserving virtual assistant by Stanford

上周我们举办了[开放家庭现状](/home-assistant/状态-of-the-open-home/)活动，其中包含了 [Genie](https://genie.stanford.edu) 的演示。

Genie 是由 [Stanford OVAL](https://oval.cs.stanford.edu/) 开发的一款开放、注重隐私的虚拟助手。在这场令人印象深刻的演示中，他们展示了 Genie 的最新能力。演示分别运行在一台刷入自定义固件的百度音箱和一台 Pi Zero 上。在这两种情况下，它都会连接到以 Home Assistant 官方插件形式运行的 Genie 服务器来完成相关功能。

<lite-youtube videoid="6ZMXE5PXPqU" videotitle="状态 of the Open Home 2021" videoStartAt="9691" posterquality="maxresdefault"></lite-youtube>

Genie 是 Almond 项目的继任者。在各种资助和赞助的支持下，斯坦福大学正在努力让 Genie 为更广泛的日常使用做好准备。

如果你想了解更多，可以查看 Genie 的[官网](https://genie.stanford.edu)以及教你如何自己动手搭建的[入门指南](https://wiki.genie.stanford.edu/getting-started/安装-guide)。如果你想和其他 Genie 用户及其开发团队交流，也可以加入他们的 [Discord](https://discord.gg/anthtR4)
或访问[社区论坛](https://community.almond.stanford.edu/)。

如果你最终在家中用上了 Genie，别忘了把你的演示和教程分享出来。
