# Updated documentation

One of the main complaints that we receive is something along the lines "I read that X is possible yet I am unable to find it on the website.". This post is to announce that we have taken the first steps to improve it by revamping the [入门] and [开发者] sections. It's still a work in progress but we now have a solid foundation to build on for the future 👍.

Our 文档 has been going through various phases. Initially it was just the README in our GitHub repository. I discovered Jekyll and GitHub pages in December 2014 and 创建 home-assistant.io. I more or less broke the README in 5 pages and [called it a website]. Back then we had a whopping 11 components!

As Home Assistant grew, so did our 文档. [Fabian Affolter](https://github.com/fabaff) does an amazing job in making sure there is at least a 文档 stub for each new feature that lands. And that's quite a feat given our [frequent releases](/home-assistant/blog/categories/发布-notes/)! But despite all the efforts, the 文档 outgrew our existing 文档 organization.

Today it has been almost 1.5 years since we started the website. We now have [264 components and platforms] under our belt and have been honored with 1.5 million page views ✨. And hopefully we now also have 文档 that our community deserves.
