title: 下一步
description: 配置 Home Assistant 的下一步
---

初始设置过程带您完成 Home Assistant 的初始设置，例如启动系统、命名您的家和选择位置。本节指向更多文档，帮助您完成下一步。

## 向 Home Assistant 添加其他人员

您可以向 Home Assistant 添加其他人员。他们可以有自己的登录，在他们的设备上使用 Home Assistant 并创建自己的仪表盘。要添加其他人员，请参阅[向 Home Assistant 添加人员](/home-assistant/integrations/person/#adding-a-person-to-home-assistant)。

## Android 和 iOS 应用

您可以在手机、智能手表甚至汽车上使用 Home Assistant。

- 要了解如何在 Android 或 iOS 上安装 Home Assistant，请参阅[Companion Apps 文档](https://companion.home-assistant.io/)。
- 想用语音控制 Home Assistant？
  - 请参阅[在 Android 上使用 Assist](/home-assistant/voice_control/android/)的文档。

## 更改基本设置

要更改位置、单位系统和语言等基本设置，请参阅[更改基本设置](/home-assistant/docs/configuration/basic/)。

## 创建备份

您可以备份 Home Assistant 配置和 Home Assistant 应用数据。如果需要回滚，备份用于恢复系统（或系统部分）。备份也用于将 Home Assistant 迁移到新硬件。在更新之前创建备份是一个好习惯。

要了解如何创建 Home Assistant 安装的备份，请参阅[创建备份](/home-assistant/common-tasks/general/#backups)的文档。

## 编辑 configuration.yaml 和配置文件访问

虽然您可以从用户界面配置大部分 Home Assistant，但对于某些集成，您需要[编辑 `configuration.yaml` 文件](/home-assistant/docs/configuration/)。此文件包含要加载的集成及其配置。在整个文档中，您会找到可以添加到配置文件以启用特定功能的代码片段。对于初学者，不需要编辑 `configuration.yaml` 文件。需要时您会被指向相关文档。

## 设置网络存储

如果您需要更多空间来存储数据，您可以配置[网络存储](/home-assistant/common-tasks/os/#network-storage)，例如用于存储备份或访问媒体。

## 开始使用语音助手

如果您想开始使用语音助手，请参阅[Assist](/home-assistant/voice_control/)的文档。
