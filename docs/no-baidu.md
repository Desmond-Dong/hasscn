# 搜索引擎

:::tip Bing

🔍 本站强烈建议你使用 [Bing](https://www.bing.com) 来搜索本站内容

:::

:::danger 拒绝百度，从我做起

🚫 本站不支持 `百度` 搜索

🚫 所有通过 `百度` 搜索以及 `百度浏览器` 访问的页面都会被定向到此页

🚫 如果你是通过 `百度` 搜索进入本站，请从上方导航栏重新开始浏览

🚫 如果你正在使用 `百度浏览器`，请更换为其他浏览器
