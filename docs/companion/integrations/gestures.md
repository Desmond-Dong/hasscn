您可以在使用 Home Assistant 主界面时设置手势以执行快速操作。

要查看和更改您的手势，请转到[设置](https://my.home-assistant.io/redirect/config/) > 伴侣应用 > 手势。对于每个手势，您可以从多个操作中选择或将手势禁用（通过将操作设置为"无"）。

:::info
对于 Android 设备：如果您的设备有可在任何应用中工作的手势，例如截屏，它们可能会覆盖 Home Assistant 手势。
:::

## 可用操作

 - **无**：禁用手势。

### Home Assistant

 - **快速搜索**：使用[快速搜索对话框](https://www.home-assistant.io/docs/tools/quick-search/)搜索任何内容。
 - **搜索实体**：使用带有实体过滤器的[快速搜索对话框](https://www.home-assistant.io/docs/tools/quick-search/)搜索实体。
 - **搜索设备**：使用带有设备过滤器的[快速搜索对话框](https://www.home-assistant.io/docs/tools/quick-search/)搜索设备。
 - **搜索命令**：使用[快速搜索对话框](https://www.home-assistant.io/docs/tools/quick-search/)命令面板搜索命令。
 - **显示侧边栏**：从任何屏幕打开侧边栏。
 - **打开 Assist**：打开 [Assist](https://www.home-assistant.io/voice_control/) 对话框。

### 导航

 - **转到上一页**：如果可用，转到上一页。
   - 仅限 iOS。在 Android 上，您可以使用 Android 的返回按钮或手势。
 - **转到下一页**：如果可用，转到下一页。
 - **转到默认仪表板**：转到为您的用户设置的默认仪表板。
   - 仅限 Android。
 - **重新加载页面**：强制重新加载当前页面。
   - 仅限 Android。在 iOS 上，当位于页面顶部时继续向下滚动以强制重新加载当前页面。

### 服务器

 - **服务器列表**：显示添加到应用的所有服务器列表，以便快速在服务器之间切换。
 - **下一个服务器**：如果您有多个服务器，激活列表中的下一个服务器。
 - **上一个服务器**：如果您有多个服务器，激活列表中的上一个服务器。

### 应用

 - **打开应用设置**：打开伴侣应用设置（设置 > 伴侣应用）。
 - **打开应用故障排除/打开调试**：打开伴侣应用的故障排除/调试选项（设置 > 伴侣应用 > 故障排除/调试）。
