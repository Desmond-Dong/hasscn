如果在阅读此处的文档后仍有问题，请尝试以下资源。

**一般帮助和配置建议**
*   访问 [Companion Apps 频道](https://discord.com/channels/330944238910963714/1284965926336335993)
*   在 [iOS 论坛](https://community.home-assistant.io/c/mobile-apps/ios) 或 [Android 论坛](https://community.home-assistant.io/c/mobile-apps/android-companion) 发帖。

**错误报告**
*   请在 [iOS GitHub 仓库](https://github.com/home-assistant/iOS) 或 [Android GitHub 仓库](https://github.com/home-assistant/android) 提交问题。

**功能请求**
* 要请求新功能，请在 Home Assistant GitHub 组织中开启讨论。使用相关平台的讨论页面：
  -  [Android 讨论页面](https://github.com/orgs/home-assistant/discussions/new?category=android)
  -  [iOS 和 macOS 讨论页面](https://github.com/orgs/home-assistant/discussions/new?category=ios-macos)。

**关于本文档的问题、更正或修改**
*   使用您想修改的页面上的编辑按钮（文档使用 [Markdown](https://daringfireball.net/projects/markdown/syntax) 编写），您需要一个 [GitHub](https://www.github.com) 帐户来提交更改；或
*   在文档的 [GitHub 仓库](https://github.com/home-assistant/companion.home-assistant) 提交问题。
