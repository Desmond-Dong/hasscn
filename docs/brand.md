# 🏷️ 直接支持接入 Home Assistant 的品牌


:::info 什么是“直接支持”？

`直接支持`指在中国地区，用户可以非常轻松、直接买到的品牌或协议设备。
部分开放协议（如 Zigbee、Z-Wave、Wi-Fi 等）涵盖品牌众多，无法一一列举
只记录开放标准可以轻松接入的，不收录需要通过复杂的逆向工程以及模拟登录等产品

:::
