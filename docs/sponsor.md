# 关于此站

此站纯是因为个人想为国内用户提供一个更简单的学习环境而搭建。非组织，非团队，非公司，非盈利。

由于个人精力有限，可能无法及时更新和维护所有内容，欢迎大家积极参与贡献和反馈。
