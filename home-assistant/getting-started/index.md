# 入门指南

感谢你为智能家居选择 Home Assistant。下面这些基础步骤将帮助你踏上通往更私密、更强大、更可持续家庭的旅程。

## 下一步

1. [安装](/home-assistant/installation/index.md) - 在设备上安装 Home Assistant。
2. [初始设置](/home-assistant/getting-started/onboarding/index.md) - 完成首次启动后的引导配置。
3. [概念和术语](/home-assistant/getting-started/concepts-terminology/index.md) - 了解实体、设备、区域等基础概念。
4. [编辑仪表盘](/home-assistant/getting-started/onboarding_dashboard/index.md) - 配置你的默认控制界面。
5. [添加集成](/home-assistant/getting-started/integration/index.md) - 连接你的设备与服务。
6. [自动化](/home-assistant/getting-started/automation/index.md) - 创建第一条让家自动运行的规则。
7. [在场检测](/home-assistant/getting-started/presence-detection/index.md) - 基于成员到家或离家触发动作。
8. [加入社区](/home-assistant/getting-started/join-the-community/index.md) - 在社区中获取帮助并分享经验。
9. [后续配置](/home-assistant/getting-started/configuration/index.md) - 继续深入你的系统设置。

## 开始前你会得到什么

* 更私密的智能家居体验
* 更强大的自动化能力
* 更灵活的本地控制方式
* 一个可以持续扩展的家庭系统
