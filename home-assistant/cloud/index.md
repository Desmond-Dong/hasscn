# Home Assistant Cloud

Home Assistant Cloud 是由我们的合作伙伴 Nabu Casa, Inc. 提供的订阅服务。请访问 [他们的网站](https://www.nabucasa.com) 了解更多关于功能、定价以及[如何配置 Home Assistant](https://www.nabucasa.com/config/)的信息。

## 功能

Home Assistant Cloud 提供以下功能：

* **远程访问** - 无需额外配置，即可安全地远程访问您的 Home Assistant 实例
* **Alexa 集成** - [连接 Amazon Alexa](/home-assistant/cloud/alexa.md) 进行语音控制
* **Google Assistant 集成** - [连接 Google Assistant](/home-assistant/cloud/google_assistant.md) 进行语音控制
* **备份存储** - 在云端存储备份
