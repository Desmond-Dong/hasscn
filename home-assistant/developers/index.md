# 开发者

Home Assistant 是一个开源项目，由全球开发者社区共同维护。我们欢迎所有形式的贡献，无论是代码、文档、翻译还是问题反馈。

## 贡献方式

### 代码贡献

1. Fork [home-assistant/core](https://github.com/home-assistant/core) 仓库
2. 创建功能分支进行开发
3. 提交 Pull Request
4. 签署 [贡献者许可协议](/home-assistant/developers/cla.md)

### 集成开发

开发新的集成是帮助 Home Assistant 支持更多设备的好方式：

* 阅读 [集成开发文档](https://developers.home-assistant.io/docs/creating_component_index)
* 遵循 [质量等级标准](/home-assistant/docs/quality_scale/index.md)
* 提交到官方集成仓库

### 文档贡献

帮助改进文档是另一种重要的贡献方式：

* Fork [home-assistant/home-assistant.io](https://github.com/home-assistant/home-assistant.io)
* 修复错误或添加新内容
* 翻译文档到其他语言
