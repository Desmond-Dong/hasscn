# 贡献者许可协议

```text
通过对本项目做出贡献，我确认：

(a) 该贡献全部或部分由我创建，我有权在 Apache 2.0 许可证下提交它；或者

(b) 该贡献基于先前的工作，据我所知，该工作在适当的开源许可证下受保护，
    我有权根据该许可证提交该工作（无论全部或部分由我创建的修改），
    在 Apache 2.0 许可证下；或者

(c) 该贡献由其他认证 (a)、(b) 或 (c) 的人直接提供给我，我未对其进行修改。

(d) 我理解并同意，本项目和贡献是公开的，贡献记录（包括我提交的所有个人信息）
    将被无限期保存，并可能与本项目或相关开源许可证一致地重新分发。
```

## 署名

本许可证的文本基于[知识共享署名-相同方式共享 3.0 未移植许可证](https://creativecommons.org/licenses/by-sa/3.0/)发布。它基于 Linux [开发者原创证书](https://elinux.org/Developer_Certificate_Of_Origin)，但经过修改以明确使用 Apache 2.0 许可证，且不提及签核。

## 签署

如果您尚未签署贡献者许可协议（CLA），并且向 Home Assistant 组织下的仓库提交拉取请求，系统将自动生成一个链接。只需按照链接及其中的说明操作即可。
