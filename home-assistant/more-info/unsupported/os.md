# 操作系统

## 问题

Home Assistant Supervisor 仅在 Home Assistant OS 上受支持。

## 解决方案

您需要使用 Home Assistant OS 重新安装运行 Supervisor 的主机。

最好的方法是[创建备份](/home-assistant/common-tasks/general/index.md#backups)您当前的安装，然后使用 Home Assistant OS 重新安装您的系统。在 [Home Assistant 入门](/home-assistant/getting-started/onboarding/index.md)期间，选择**从备份恢复**。
