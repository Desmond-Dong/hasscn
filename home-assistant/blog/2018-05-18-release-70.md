# 0.70：Facebox、SpaceAPI、Konnected 报警系统

0.70 版本来了。因为前端构建方式做了大规模重构，这次发布时间比预期稍晚一些。它不应该影响前端的外观和使用体验。感谢所有参与测试版并帮助验证的朋友。目前看 Firefox 在运行 Hass.io 面板时仍有一些问题，我们正在尽快发布修复。

如果你正在使用自定义 UI 或面板，这个版本有一些变更。请确保升级到最新版本。更多信息请查看我们的[开发者博客](https://开发者.home-assistant.io/博客/2018/05/22/custom-ui-面板-api.html)。

说到开发者博客，我们还上线了一个新的博客和配套的 [Twitter 账号](https://twitter.com/hass_devs)。这是我们持续推进的一部分：把 Home Assistant 用户与开发者的内容和互动渠道更清晰地分开。

这个版本里一个很酷的新组件是 [Konnected][konnected docs]（[产品页](https://konnected.io/)）。它可以让你把现有的有线报警系统直接接入 Home Assistant！

另一个很棒的新功能是 [Facebox][image_processing.facebox docs]。它让你可以在本地对摄像头画面进行人脸检测。很期待大家用它玩出新花样。

## 不向后兼容的变更

* SABnzbd 已转换为组件。请使用以下新配置进行设置：
  ```yaml
  sabnzbd:
    api_key: YOUR_SABNZBD_API_KEY
  ```
  ([@jeradM] - [#12915]) ([sabnzbd docs]) ([传感器.sabnzbd docs]) (breaking change) (new-platform)
* 对于 BMW 电动车：当前传感器 `remaining_range_fuel` 已被 `remaining_range_total` 取代，用于表示带燃油发动机车辆的总剩余续航。([@gerard33] - [#14293]) ([bmw\_connected\_drive docs][bmw_connected_drive docs]) ([binary\_sensor.bmw\_connected\_drive docs][binary_sensor.bmw_connected_drive docs]) ([传感器.bmw\_connected\_drive docs][传感器.bmw_connected_drive docs]) (breaking change)
* 由于 simplepush.io 网站已离线，相关应用也已无法使用，因此该服务已被移除。([@c727] - [#14358]) ([通知 docs]) (breaking change)
* DarkSky 传感器：新增体感温度高/低值，废弃原先的 max/min。([@nordlead2005] - [#12233]) ([传感器.darksky docs]) (breaking change)
