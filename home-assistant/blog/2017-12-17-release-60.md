# 0.60: Beckhoff/TwinCAT, WebDav, Gearbest, iAlarm

<a href='/home-assistant/integrations/#版本/0.60'><img src='/home-assistant/images/blog/2017-12-0.60/components.png' style='border: 0;box-shadow: none;'></a>

0.60 最大的变化会在另一篇单独的[博客文章](/home-assistant/blog/2017/12/17/introducing-home-assistant-cloud/)中介绍，所以这里我们就简短一些。只提一句：这是 2017 年最后一个版本。2018 年我们会恢复双周发布节奏。

非常感谢所有帮助我们促成这次发布的人。

## TwinCAT

借助 [@stlehmann] 带来的全新 [ADS（自动化设备规范）][ads docs] 组件，你现在可以将 Home Assistant 接入这种独立于现场总线的接口，而它通常用于运行 TwinCAT 的 Beckhoff 设备之间。

## WebDav 日历

感谢 [@maxlaverse]，Home Assistant 现在已经支持 [WebDav 日历][日历.caldav docs]。

## 价格追踪

借助新的 `gearbest` 传感器，现在又多了一个可以追踪商品价格的传感器可供使用。

## 金融信息

Yahoo! 已经停止了他们的金融服务。为了填补这一空缺，我们在这次发布中加入了新的 [`alpha_vantage` 传感器][传感器.alpha_vantage docs]，让你可以继续监控股票市场。
