    #总结“到底是什么？！”这个月

    一个月前，我写了一篇博客文章，开始了我们的第一个月 “什么鬼？！” （WTH）曾经！一次实验，没有任何期望， 因为我们以前从未做过这样的事情。 But WTH, why not. 😁

    今天，我想我们实际上可以说：一切美好的事情都会结束。

    这是令人惊奇的一个月！已创建多达 746 个主题 [WTH 论坛][论坛]，总共收集了超过 4,500 个帖子。的内容 all these 创建 topics are amazing! And apparently, you all agreed on that, 所有这些帖子都获得了惊人的 2,600 个赞。

    论内容，真是太精彩了！那里有一些非常好的反馈。 烦恼让很多人想：“是啊！什么鬼？！”更重要的是， 开发者也插话了！

    开发人员投入并开始解决和修复问题！眼下 在编写过程中，有多达 68 个 Pull 请求，其中包含代码更改以解决这些 WTH 已经做了。其中很多已经随 0.115 生日 发布.

    #那么，修复了什么？

    让我们快速浏览一下已报告和修复的内容/addressed/improved （排名不分先后）：

    • Duplicate automation Button
    • Manage 侧边栏/menu items (hide/add/move)
    • Stop requiring magic to enable templates
    • WTH can I not use the 更新 button while running an NGINX HA proxy
    • How the heck can I set this user picture?
    • Why the heck isn’t there any official macOS desktop app?!
    • What the heck turned my 灯光 on?
    • 重载 integrations individually
    • Why doesn’t a 触发器 platform for 状态 属性 exist?
    • WTH why can’t I permanently delete jinja2 教程 in the template editor
    • WTH why can’t an administrator change a user’s password
    • WTH: Can’t we see which 触发器 fired an automation?
    • Set static IP in supervisor
    • WTH: Why can’t I use “for” without defining a “to”-状态 in 自动化?
    • Option to template the keys in data_template in addition to only values
    • WTH Slack no username/icon customization
    • Dev Tools –> Template – WTH isn’t it refreshing automatically?!
    • Why the heck is a 重启 needed for each an every change to 配置.yaml? （部分解决，已经取得了很多进展）
    • WTH do pressing 更新 button doesn’t have confirmation
    • WTH can’t I click outside of a dialog to close the dialog?

    这个列表并不完整，但很好地说明了所发生的事情。我们认为 这是一项令人印象深刻的成就。如果你对 GitHub 有一点了解，你可以找到 WTH 月份提出的所有拉取请求，因为它们都已标记 与“WTH”。

    #WTH 论坛和 Hacktoberfest 闭幕

    正如所说，一切美好的事情都会结束。 WTH 论坛即将结束。 这意味着无法创建新的 WTH 主题，但所有内容都将保留 可见的。

    Hacktoberfest 即将来临，该论坛中的所有当前数据 确实可以帮助驱动它。对于开发人员来说，这是一个很好的资源 在十月期间做出贡献的机会。所以，当我们关闭时 下来，我们希望更多的内容能够得到解决。

    为什么不一直运行这个 WTH 呢？嗯，随着时间的推移，它会失去力量。这是 当比较开始和结束的速度时现在已经可见 当月的。

    不过，让我们看看明年是否可以再做一次。或许，我们应该结合起来 2021 年 Hacktoberfest 吗？

    WTH这个月你过得怎么样？我们应该再做一次吗？
