    Home Assistant 12 岁

    #Home Assistant 0.41: Tado, Volumio, Workday, improved Plex

    欢迎来到 0.41。在过去两周里，发生了很多事情，不只是代码层面，Home Assistant 的社区层面也同样热闹。Paulus 接受了 OpenSourceCraft 的采访，Fabian 在 Chemnitzer Linux Tage 上举办了工作坊，而且我们现在已经是一个获奖的开源项目了（我会在另一篇博客文章里专门介绍这件事）。

    #Plex

    @JesseWebDotCom 对 Plex 媒体播放器平台进行了大幅改进，包括更好的元数据支持、新的配置选项，以及更完善的控制能力和对不可控客户端的处理。

    #组件概览

    Components 概览页面现在支持搜索和筛选功能，可以更快找到你需要的组件或平台。再次感谢 @bdurrer 的贡献。

    #更新日志

    从 0.40 开始启用的新更新日志格式，会附上对应的 pull request 链接。发布说明不会覆盖所有改动，但我们认为这个补充能让你更容易查到每项变更的详细信息。

    #新平台与组件

    • 新增 Tado 温控设备支持 (@wmalgadey)
    • 新增 Volumio 媒体播放器 (@jslove)
    • 新增 Workday 传感器 (@BastianPoe)

    #不向后兼容的变更

    • Kodi notifier 平台已迁移到 async，并与 Kodi 媒体播放器平台的配置保持一致 (#6497)。
    • Music Player Daemon（MPD）平台中的 location 已替换为 name (#6553)。
    • 事件装饰器已被移除 (#6634)。
    • Emby 媒体播放器 平台已调整，以避免名称冲突 (#6664)。
    • 多处功率和能量单位已更新，这项变更主要影响 switch 平台 (#6212)。
    • 如果设置为 auto，MQTT 将自动使用内置证书 (#6707)。
    • Android IP Webcam 的自动发现功能已移除 (#6528)。
    • 前端现在改用 Shadow DOM，这可能会影响你的自定义面板 (#228)。

    #如果你需要帮助

    欢迎使用我们非常活跃的 论坛，或者加入聊天频道 discord。发布说明虽然也开放评论，但我们更推荐你使用前面的沟通渠道。谢谢。

    #反馈问题

    如果你遇到了这次发布引入的问题，请到我们的 问题追踪器 提交反馈，并确保填写问题模板中的所有字段。
