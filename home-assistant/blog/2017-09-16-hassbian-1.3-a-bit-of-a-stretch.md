# Hassbian 1.3 - A bit of a stretch

This new 发布 finally brings us the 发布 of Hassbian based on the 发布 of Raspbian Stretch. The most significant change with this is that we now run Home Assistant on Python 3.5.
Other than that the changes are mostly to our tool `hassbian-config`.

### Hassbian-config

To allow you to customize your 安装 further, we have made a few additions to our tool called `hassbian-config`. Majority of these changes where contributed by [@Ludeeus][ludeeus].

#### 升级 脚本

[@Ludeeus][ludeeus] contributed the possibility to run 升级 脚本 to the `hassbian-config` tool.

* 升级 Hassbian: Upgrades packages of the base operating system. Contributed by [@Landrash][landrash].
* 升级 Hassbian-脚本: Downloads and upgrades the latest 发布 of Hassbian-脚本. Contributed by [@Ludeeus][ludeeus].
* 升级 Home-assistant: Upgrades the Home Assistant 安装 to the latest 发布. Contributed by [@Ludeeus][ludeeus].
