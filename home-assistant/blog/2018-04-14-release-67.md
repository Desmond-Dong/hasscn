# 0.67：Mastodon、Tahoma 开关与 Nanoleaf Aurora 灯光面板

<a href='/home-assistant/integrations/#版本/0.67'><img src='/home-assistant/images/blog/2018-04-0.67/components.png' style='border: 0;box-shadow: none;'></a>

0.67 版本来了！如果你错过了[昨天的公告][ann-ubiquiti]：Ubiquiti 现在正在赞助我撰写这些发布说明。没错，就是这么棒。这样我就有更多时间专注于我们推迟已久的一些重大改进，首先会是用户系统。

这个版本包含一项安全修复。在此前版本中，错误日志可通过 API 访问且不要求认证。虽然这本身不一定直接导致泄露，但如果某个组件错误地将敏感信息写入错误日志，就可能造成暴露（目前我们并不知道有组件会这样做）。感谢 [Matt Hamilton](https://github.com/Eriner) 向我们披露这个问题。
