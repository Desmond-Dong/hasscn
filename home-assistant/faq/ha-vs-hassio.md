# Home Assistant 与 Home Assistant Core 的区别

简单来说，Home Assistant Core 是一个 Python 程序。它可以在各种操作系统上运行，并提供跟踪、控制和自动化设备的能力。
当人们谈论 Home Assistant Core 时，通常指的是独立[安装类型](/home-assistant/docs/installation/)。

[Home Assistant](/home-assistant/hassio/) 是 Home Assistant Core 和工具的组合，可以让您轻松地在树莓派和其他平台上运行，无需先设置操作系统。
Home Assistant 是一个一体化解决方案，具有可从 Home Assistant 前端访问的管理用户界面。此界面在 Home Assistant Core 设置中不存在。

请注意，应用程序仅在常规 Home Assistant 安装中可用。
