# 组织

Home Assistant 提供多种方式来组织您的智能家居设备和实体。

## 组织方式

* [区域](/home-assistant/docs/organizing/areas.md) - 按房间或位置分组设备
* [楼层](/home-assistant/docs/organizing/floors.md) - 按楼层组织区域
* [标签](/home-assistant/docs/organizing/labels.md) - 使用标签分类实体
* [分类](/home-assistant/docs/organizing/categories.md) - 按类型对项目分组

## 界面功能

* [表格](/home-assistant/docs/organizing/tables.md) - 使用表格管理项目

## 组织优势

良好的组织结构可以：

* 更轻松地管理设备
* 在语音助手中更自然地控制设备
* 在自动化中更方便地引用设备组
* 在仪表盘中更好地展示设备
