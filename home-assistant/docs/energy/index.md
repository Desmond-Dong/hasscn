# 能源

Home Assistant 能源仪表板让您能够监控和管理家中的能源使用情况。

## 能源类型

* [电网](/home-assistant/docs/energy/electricity-grid.md) - 配置电力监控
* [燃气](/home-assistant/docs/energy/gas.md) - 配置燃气监控
* [水](/home-assistant/docs/energy/water.md) - 配置水用量监控

## 可再生能源

* [太阳能电池板](/home-assistant/docs/energy/solar-panels.md) - 配置太阳能发电
* [电池](/home-assistant/docs/energy/battery.md) - 配置储能电池

## 设备监控

* [单个设备](/home-assistant/docs/energy/individual-devices.md) - 监控单个设备的能耗

## 故障排除

* [常见问题](/home-assistant/docs/energy/faq.md) - 能源面板常见问题解答

## 能源面板功能

能源面板可以：

* 可视化能源生产和消耗
* 计算自给率和成本
* 跟踪历史数据
* 集成电价信息
