# Home Assistant Hardware

**Home Assistant Hardware** 集成提供以下设备之间共享的功能：[Home Assistant Connect ZBT-2](/home-assistant/integrations/homeassistant_connect_zbt2/index.md)、[Home Assistant Connect ZBT-1](/home-assistant/integrations/homeassistant_sky_connect/index.md)、[Home Assistant Green](/home-assistant/integrations/homeassistant_green/index.md) 和 [Home Assistant Yellow](/home-assistant/integrations/homeassistant_yellow/index.md)。

## Configuration

此集成不提供用户可配置项。
