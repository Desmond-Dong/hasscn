# Hunter Hydrawise

**Hunter Hydrawise** 集成允许您将 [Hunter Hydrawise](https://hydrawise.com) Wi-Fi 灌溉控制系统集成到 Home Assistant 中。

目前 Home Assistant 支持以下设备类型：

* [二值传感器](#二值传感器)
* [传感器](#传感器)
* [开关](#开关)
* [阀门](#阀门)

## 前提条件

1. 要设置 Hydrawise 集成，您必须先在 <https://app.hydrawise.com> 创建一个在线账户。
2. 然后，使用您的在线凭据配置集成。

:::note 配置
此集成可通过 UI 配置。前往 **设置 > 设备与服务** 添加。
:::

## 二值传感器

为控制器创建二值传感器实体：

* 云 API 可用性
* 雨量传感器湿度检测（如果控制器连接了雨量传感器）

为每个灌溉区域创建二值传感器实体：

* 运行状态

## 传感器

为控制器创建传感器实体：

* 每日总用水量（如果控制器连接了流量传感器）
* 每日活动用水量（如果控制器连接了流量传感器）
* 每日非活动用水量（如果控制器连接了流量传感器）

为每个灌溉区域创建传感器实体：

* 下一次计划自动浇水周期的时间戳
* 当前浇水周期的剩余时间（分钟）
* 区域的每日活动用水量（如果灌溉区域连接了流量传感器）

## 开关

为每个区域添加开关，控制：

* 智能浇水功能
* 手动浇水

### 开关操作

当 `auto_watering` 为 `on` 时，灌溉区域将遵循通过 Hydrawise [移动或网络应用程序](https://www.hydrawise.com)设置的智能浇水计划。当 `auto_watering` 开关为 `off` 时，区域的智能浇水计划将暂停 1 年。

当 `manual_watering` 为 `on` 时，区域将运行 15 分钟。

:::note
由于 Hydrawise API 的更改，自动浇水开关的状态已更改。在正常条件下，自动浇水开关正确反映 Hydrawise 移动或网络应用程序上的智能浇水计划。但是，如果系统连接了雨量传感器且处于活动状态（检测到雨水），或区域正在运行，自动浇水开关将关闭。当这两个条件都消除后，开关将再次显示正确的自动浇水状态。

:::

## 动作

Hydrawise 集成提供各种自定义动作。

### 动作：开始浇水

`hydrawise.start_watering` 动作在灌溉区域启动浇水周期。

| 数据属性 | 可选 | 描述                                                                                                                                                                                     |
| -------------- | -------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `entity_id`    | 否       | 要开始浇水的灌溉区域。                                                                                                                                                 |
| `duration`     | 是      | 运行浇水周期的时间长度（分钟）。如果未指定（或为零），将使用 Hydrawise 移动或网络应用程序中为灌溉区域设置的默认浇水持续时间。 |

### 动作：暂停

`hydrawise.suspend` 动作暂停灌溉区域的自动浇水直到指定日期。

| 数据属性 | 可选 | 描述                                                                                |
| -------------- | -------- | ------------------------------------------------------------------------------------------ |
| `entity_id`    | 否       | 要暂停浇水的灌溉区域。                                          |
| `until`        | 否       | 自动浇水应恢复的日期和时间。例如，`2024-08-30 08:30` |

### 动作：恢复

`hydrawise.resume` 动作恢复灌溉区域的自动浇水。

| 数据属性 | 可选 | 描述                                      |
| -------------- | -------- | ------------------------------------------------ |
| `entity_id`    | 否       | 要恢复浇水的灌溉区域。 |

## 阀门

为每个区域添加一个阀门，允许手动控制区域浇水。

当通过 Home Assistant 打开区域的阀门时，它将设置自动关闭时间为 Hydrawise [移动或网络应用程序](https://www.hydrawise.com)中配置的区域默认浇水持续时间。
