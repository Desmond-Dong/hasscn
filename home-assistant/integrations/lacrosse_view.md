# LaCrosse View

[**LaCrosse View**](https://www.lacrossetechnology.com/pages/la-crosse-view) 是 LaCrosse Technology 提供的服务，可让您访问 LaCrosse 传感器。

它在欧洲也以 [**TFA View**](https://www.tfa-dostmann.de/en/produkte/weather-stations/wifi-weather-stations/) 的名称提供。

:::tip
如果您正在寻找 [**Jeelink LaCrosse sensors**](/home-assistant/integrations/lacrosse.md) 的集成，请查看[**这里**](/home-assistant/integrations/lacrosse.md)。

:::
:::note 配置
此集成可通过 UI 配置。前往 **设置 > 设备与服务** 添加。
:::

## 支持的传感器

此集成支持 [**此页面**](https://www.lacrossetechnology.com/collections/lacrosse-view-connected) 上列出的所有传感器。
