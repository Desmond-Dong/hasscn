# Zooz

[Zooz](https://www.getzooz.com/) 的 Z-Wave 设备可在本地运行，并与 Home Assistant 中的 Z-Wave 集成无缝配合使用（需要 Z-Wave 棒）。由于所有连接都在本地完成，因此状态更新和设备控制在 Home Assistant 中几乎是即时的。

[![Open Add device in your Home Assistant instance.](https://my.home-assistant.io/badges/add_zwave_device.svg)](https://my.home-assistant.io/redirect/add_zwave_device/?brand=zooz)

[了解有关 Home Assistant 中 Z-Wave 的更多信息。](/home-assistant/integrations/zwave_js/index.md)
