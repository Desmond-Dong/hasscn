# Govee Bluetooth

将 [Govee](https://www.govee.com/) BLE 设备集成到 Home Assistant 中。

:::note 配置
此集成可通过 UI 配置。前往 **设置 > 设备与服务** 添加。
:::

启用并正常运行 [Bluetooth](/home-assistant/integrations/bluetooth.md) 集成后，Govee BLE 集成会自动发现设备。
