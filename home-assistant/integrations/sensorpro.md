# SensorPro

Integrates [SensorPro](https://www.sigmawit.com/) devices into Home Assistant.

## Supported devices

* T201
* T301

The SensorPro integration will automatically discover devices once the [Bluetooth](/home-assistant/integrations/bluetooth.md) integration is enabled and functional.

:::note 配置
此集成可通过 UI 配置。前往 **设置 > 设备与服务** 添加。
:::
