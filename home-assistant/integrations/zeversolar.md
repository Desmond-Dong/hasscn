# Zeversolar

**Zeversolar** 集成使用设备的本地 IP 来获取当前功率和今日总发电量等信息。
此集成可让您收集并保存数据，从而查看 Zeversolar 发电情况的历史概览。

## 前提条件

您需要一台已连接到太阳能板和 Wi-Fi 的 Zeversolar 逆变器。此外，您还需要获取该 Zeversolar 设备的 IP 地址。

:::note 配置
此集成可通过 UI 配置。前往 **设置 > 设备与服务** 添加。
:::

## 传感器类型

配置完成后，此集成会为每台已配置的逆变器创建两个传感器：

* 今日发电量，单位为 `kWh`（千瓦时）
* 当前功率，单位为 `W`（瓦）
