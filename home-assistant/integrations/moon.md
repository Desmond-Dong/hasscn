# Moon

The **Moon** integration tracks the phases of the moon.

:::note 配置
此集成可通过 UI 配置。前往 **设置 > 设备与服务** 添加。
:::

The sensor provided by this integration will return one of the following values:

* `new_moon`
* `waxing_crescent`
* `first_quarter`
* `waxing_gibbous`
* `full_moon`
* `waning_gibbous`
* `last_quarter`
* `waning_crescent`
