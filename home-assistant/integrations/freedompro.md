# Freedompro

[Freedompro](https://freedompro.eu/) 是一家专门从事家庭自动化的公司，设计和制造产品，使每个人、安装人员和爱好者都能负担得起家庭自动化产品。 [Freedompro Products](https://freedompro.eu/collections/easykon) 的设计易于使用、实用且安装快速。

Freedompro 及其安装在电气系统上的产品使您可以控制家中已有的所有 KNX 和 MyHome SCS 设备。

通过这种集成，您可以控制所有 [Freedompro](https://freedompro.eu/) 配件。

## 初步步骤

您需要从 Freedompro 的应用程序获取 API 密钥才能使用此集成。

1. 在手机上下载应用程序：([Android](https://play.google.com/store/apps/details?id=eu.freedompro.app) / [iOS](https://apps.apple.com/app/id1598082027))。
2. 注册并创建您的第一个家。
3. 在开始集成之前，请确保您至少有一个与 Freedompro Cloud 配对的配件。
4. 切换到**个人资料**选项卡（右下角）。
5. 向下滚动到**开发者工具**并点击**Api Key**。
6. 生成 API 密钥。

:::注意配置
此集成可通过 UI 配置。前往 **设置 > 设备与服务** 添加。
:::
