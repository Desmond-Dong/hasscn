# Ubisys

[Ubisys](https://www.ubisys.com/) 致力于确保其产品保持最新状态，并可在 Home Assistant 中直接使用。

Ubisys Zigbee 设备可在本地运行，并能与 Home Assistant 中的 Zigbee 集成无缝配合。由于所有连接都在本地完成，因此设备状态更新和控制操作都能立即在 Home Assistant 中生效。
