# My Home Assistant

此集成用于处理来自 [My Home Assistant](https://my.home-assistant.io) 服务的重定向。

My Home Assistant 允许文档直接链接到您 Home Assistant 实例中的特定页面。更多信息请参阅 [My Home Assistant 常见问题](https://my.home-assistant.io/faq/)。

## 配置

除非您在配置中禁用了或移除了 [`default_config:`](/home-assistant/integrations/default_config/index.md)，否则此集成默认处于启用状态。如果确实如此，下面的示例展示了如何手动启用此集成：

将以下内容添加到您的 `configuration.yaml` 文件中：

```yaml
# Example configuration.yaml entry
my:
```
