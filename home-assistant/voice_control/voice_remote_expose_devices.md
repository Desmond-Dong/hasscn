# 向 Assist 暴露实体

如果您想通过语音命令控制设备，必须先将对应的实体暴露给 Assist。
这样做是为了避免门锁、车库门等敏感设备被语音误操作。

## 将设备暴露给 Assist

1. 前往 **设置** > **语音助手**。
2. 打开 **暴露** 选项卡。
   ![暴露实体选项卡](/home-assistant/images/assist/assistant-expose-01.png)
3. 如果要控制某个特定实体的设置，请从列表中选择该实体。
   * 在弹出窗口中，选择要将该实体暴露给哪些助手：Assist、Google Assistant 和/或 Alexa。
     ![暴露实体选项卡](/home-assistant/images/assist/assistant-expose-10.png)
4. 如果要一次性将多个实体暴露给所有助手，请选择 **暴露实体** 按钮。
5. 选择所有您希望能够通过语音控制的实体。
   ![暴露实体选项卡](/home-assistant/images/assist/assistant-expose-02.png)
